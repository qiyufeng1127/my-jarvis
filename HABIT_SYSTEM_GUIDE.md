# 习惯追踪系统使用指南

## 功能概览

习惯追踪系统提供了三种方式来管理你的习惯：

### 1. 手动添加习惯 ✍️

点击"添加习惯"按钮，手动配置习惯的各项参数。

### 2. AI 智能填写 🤖

输入习惯名称后，点击"智能填写"按钮，AI 会自动推荐：
- 合适的 emoji 图标
- 习惯类型（计数/时长/布尔）
- 建议的频率（日/周/月/年）
- 目标值
- 识别关键词
- 反向检测规则（如适用）

**示例：**
- 输入"跑步" → AI 推荐：🏃 时长型，每天30分钟
- 输入"喝水" → AI 推荐：💧 计数型，每天8次
- 输入"不熬夜" → AI 推荐：🌙 布尔型，反向检测规则（23:00-05:00 未找到"工作"等关键词）

### 3. AI 自动识别 🎯

系统每天自动分析时间轴上的任务，识别潜在的习惯模式：
- **日习惯**：连续3天每天都做
- **周习惯**：连续2周每周做3次以上
- **月习惯**：连续2月每月做8次以上

识别到的习惯会显示在顶部的"AI 发现的习惯候选"区域，你可以选择接受或忽略。

## 核心功能

### 目标模式

**按频率模式**
- 每天/周/月完成指定次数
- 适合：喝水、吃水果、运动等

**按时长模式**
- 累计时长达到目标即可
- 适合：阅读、学习、睡眠等
- 自动累加任务的实际时长

### 自动识别规则

#### 正向识别
设置关键词，当任务标题或标签包含这些关键词时，自动记录到习惯。

**示例：**
- 习惯"跑步"，关键词：跑步、晨跑、夜跑
- 完成任务"晨跑30分钟" → 自动记录到"跑步"习惯

#### 反向检测 🌙
在指定时间段内**未找到**关键词则算完成，适合"不做某事"的习惯。

**示例：不熬夜**
- 检测时间：23:00 - 次日 05:00
- 反向关键词：工作、学习、加班、熬夜、刷手机
- 如果在这个时间段内没有包含这些关键词的任务 → 算作"不熬夜"完成 ✅

**配置步骤：**
1. 开启"AI 自动识别"
2. 开启"反向检测规则"
3. 设置检测时间段（开始时间、结束时间）
4. 勾选"跨天"（如果时间跨越午夜）
5. 添加反向关键词（未找到这些词则算完成）

### 统计分析

每个习惯都有详细的统计数据：
- **当前连续**：当前连续完成的天数 🔥
- **最长连续**：历史最长连续天数
- **总次数**：累计完成次数
- **完成率**：本周/本月完成率
- **日历热力图**：可视化完成情况

### 快速打卡

在习惯列表中，点击右侧的"+"按钮即可快速打卡。

## 使用场景

### 场景 1：培养运动习惯
1. 添加习惯"跑步"
2. 使用 AI 智能填写 → 自动配置为时长型，每天30分钟
3. 设置关键词：跑步、晨跑、夜跑
4. 每次完成跑步任务，自动记录到习惯

### 场景 2：改善睡眠
1. 添加习惯"不熬夜"
2. 使用 AI 智能填写 → 自动配置反向检测
3. 时间段：23:00 - 05:00
4. 反向关键词：工作、学习、加班、熬夜
5. 系统每天凌晨自动检查，如果没有这些关键词的任务 → 打卡成功

### 场景 3：保持学习
1. 添加习惯"阅读"
2. 按时长模式，每天30分钟
3. 设置关键词：阅读、读书、看书
4. 完成相关任务自动累计时长

## 规则配置

点击右上角的设置图标，可以调整 AI 识别的阈值：

- **日习惯阈值**：连续N天 → 生成日习惯候选
- **周习惯阈值**：连续N周，每周至少M次
- **月习惯阈值**：连续N月，每月至少M次

## 技巧提示

1. **善用 AI 智能填写**：输入习惯名称后，先点击"智能填写"，再根据需要微调
2. **关键词要精准**：避免过于宽泛的关键词，以免误匹配
3. **反向检测适合"不做"类习惯**：不熬夜、不吃零食、不刷手机等
4. **查看候选习惯**：定期查看 AI 发现的习惯候选，可能有意外收获
5. **利用统计数据**：通过日历热力图和完成率，了解自己的习惯养成情况

## 常见问题

**Q: 反向检测什么时候执行？**
A: 每天凌晨1点自动执行，检查前一天的情况。

**Q: 如何修改已有习惯的规则？**
A: 点击习惯卡片进入详情页，可以编辑所有配置。

**Q: AI 识别的习惯不准确怎么办？**
A: 可以在设置中调整识别阈值，或者直接忽略不需要的候选习惯。

**Q: 可以同时使用正向和反向检测吗？**
A: 不建议。一个习惯应该只使用一种检测方式。
