## 🛒 第二步：购买域名（阿里云示例）

### 方式一：阿里云（推荐）

#### 1. 注册阿里云账号
- 访问：https://www.aliyun.com/
- 点击右上角"免费注册"
- 使用手机号注册（需要实名认证）

#### 2. 进入域名注册页面
- 访问：https://wanwang.aliyun.com/
- 或在阿里云首页搜索"域名注册"

#### 3. 查询域名是否可用
```
在搜索框输入：manifestos
系统会显示各种后缀的可用情况：
✅ manifestos.cn - 可注册
✅ manifestos.com - 可注册
❌ manifestos.net - 已被注册
```

#### 4. 加入购物车并结算
```
1. 选择你想要的域名（如 manifestos.cn）
2. 选择购买年限（建议 1-3 年）
3. 点击"加入清单"
4. 进入购物车结算
5. 支付（支持支付宝/微信）
```

#### 5. 实名认证（必须）
```
购买后需要实名认证：
1. 进入"控制台" → "域名"
2. 找到你的域名，点击"实名认证"
3. 上传身份证照片
4. 等待审核（通常 1-24 小时）
```

**重要提示：**
- ⚠️ 域名必须实名认证后才能使用
- ⚠️ .cn 域名必须是中国大陆实名认证
- ⚠️ 实名认证通过前，域名无法解析

## ⚙️ 第三步：在 Vercel 添加域名

### 详细步骤（带截图说明）

#### 1. 登录 Vercel
```
访问：https://vercel.com/
使用 GitHub 账号登录
```

#### 2. 进入你的项目
```
1. 在 Dashboard 找到你的项目（manifestos）
2. 点击项目名称进入
```

#### 3. 进入域名设置
```
1. 点击顶部导航栏的 "Settings"
2. 在左侧菜单找到 "Domains"
3. 点击进入域名管理页面
```

#### 4. 添加域名
```
在 "Add Domain" 输入框中输入你的域名：

选项 A：使用 www 子域名（推荐）
输入：www.manifestos.cn

选项 B：使用根域名
输入：manifestos.cn

选项 C：两个都添加（最佳）
先添加：www.manifestos.cn
再添加：manifestos.cn
```

#### 5. 获取 DNS 配置信息
```
添加域名后，Vercel 会显示需要配置的 DNS 记录：

示例（www 子域名）：
类型: CNAME
名称: www
值: cname.vercel-dns.com

示例（根域名）：
类型: A
名称: @
值: 76.76.21.21
```

**重要：记下这些信息，下一步要用！**

## 🔧 第四步：配置 DNS（阿里云示例）

### 在阿里云配置 DNS

#### 1. 进入域名控制台
```
1. 登录阿里云
2. 进入"控制台"
3. 点击左侧菜单"域名"
4. 找到你的域名（如 manifestos.cn）
```

#### 2. 进入 DNS 解析设置
```
1. 点击域名右侧的"解析"按钮
2. 进入解析设置页面
```

#### 3. 添加解析记录

**方案 A：只使用 www 子域名（推荐新手）**

```
点击"添加记录"，填写：

记录类型：CNAME
主机记录：www
解析线路：默认
记录值：cname.vercel-dns.com
TTL：10分钟

点击"确认"保存
```

**方案 B：同时配置根域名和 www（推荐）**

```
第一条记录（www 子域名）：
记录类型：CNAME
主机记录：www
解析线路：默认
记录值：cname.vercel-dns.com
TTL：10分钟

第二条记录（根域名）：
记录类型：A
主机记录：@
解析线路：默认
记录值：76.76.21.21
TTL：10分钟

点击"确认"保存
```

#### 4. 配置域名跳转（可选）
```
如果你想让 manifestos.cn 自动跳转到 www.manifestos.cn：

记录类型：显性URL
主机记录：@
记录值：http://www.manifestos.cn
```

## ⏱️ 第五步：等待生效

### DNS 生效时间

```
⏰ 通常需要：10-30 分钟
🌍 完全生效：最多 24-48 小时

检查方法：
1. 在浏览器输入你的域名
2. 如果能打开网站，说明已生效
3. 如果显示错误，继续等待
```

### 验证 DNS 是否生效

#### 方法 1：在线工具
```
访问：https://tool.chinaz.com/dns/
输入你的域名，点击查询
如果显示 Vercel 的 IP，说明配置成功
```

#### 方法 2：命令行（Windows）
```
打开 CMD，输入：
nslookup www.manifestos.cn

如果返回 Vercel 的 IP，说明成功
```

#### 方法 3：直接访问
```
在浏览器输入：
http://www.manifestos.cn

如果能打开你的网站，说明成功！
```

## 🔍 常见问题解决

### 问题 1：域名无法访问

**可能原因：**
- DNS 还未生效（等待 10-30 分钟）
- DNS 配置错误（检查记录值）
- 域名未实名认证（完成认证）

**解决方法：**
```
1. 检查 DNS 配置是否正确
2. 使用 nslookup 验证 DNS
3. 等待更长时间（最多 24 小时）
```

### 问题 2：显示 "Invalid Configuration"

**解决方法：**
```
1. 检查 CNAME 记录值是否为：cname.vercel-dns.com
2. 检查主机记录是否正确（www 或 @）
3. 删除域名重新添加
```

### 问题 3：HTTPS 不生效

**解决方法：**
```
1. 等待 10-30 分钟
2. 在 Vercel 项目设置中检查 SSL 状态
3. 如果长时间不生效，联系 Vercel 支持
```

### 问题 4：中国仍然无法访问

**可能原因：**
- 域名本身被墙（极少见）
- DNS 污染

**解决方法：**
```
1. 更换 DNS 服务器（使用 DNSPod）
2. 使用 Cloudflare CDN
3. 考虑使用国内部署平台
```
