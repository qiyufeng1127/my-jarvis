## 📋 你需要完成的步骤

### ⏳ 步骤 3: 设置 Supabase 数据库

**操作步骤**:

1. **访问 Supabase 控制台**
   ```
   https://supabase.com/dashboard/project/nucvylmszllecoupjfbh
   ```

2. **进入 SQL Editor**
   - 点击左侧菜单的 **SQL Editor**
   - 点击 **New Query**

3. **执行数据库架构**
   - 打开项目中的 `supabase/schema.sql` 文件（已在你的编辑器中打开）
   - 全选（Ctrl+A）并复制（Ctrl+C）
   - 粘贴到 SQL Editor（Ctrl+V）
   - 点击 **Run** 按钮（或按 Ctrl+Enter）
   - 等待执行完成（约 10-30 秒）

4. **验证数据库**
   - 点击左侧菜单的 **Table Editor**
   - 应该看到 13 个表：
     - users
     - tasks
     - growth_dimensions
     - long_term_goals
     - identity_levels
     - bad_habits
     - bad_habit_occurrences
     - gold_transactions
     - growth_history
     - sync_logs
     - reward_store
     - reward_redemptions
     - achievements

### ⏳ 步骤 6: 创建第一个任务

**操作步骤**:

1. 点击"开始我的成长之旅"按钮
2. 进入主控面板
3. 点击"添加任务"按钮
4. 填写任务信息：
   - 标题：输入任务名称
   - 类型：选择任务类型
   - 优先级：选择 1-4
   - 时长：选择预计时间
5. 点击"创建任务"
