## 💵 获得金币

### 方式 1: 完成任务（主要来源）

#### 金币计算公式

```
总金币 = 基础奖励 × 时长系数 × 优先级系数 × 连击系数
```

#### 基础奖励（按任务类型）

| 任务类型 | 基础金币 | 说明 |
|---------|---------|------|
| 💼 工作任务 | 100 | 工作相关 |
| 📚 学习任务 | 80 | 学习提升 |
| 💪 健康任务 | 60 | 运动健康 |
| 🎨 创作任务 | 90 | 创作产出 |
| 🏠 生活任务 | 40 | 日常琐事 |
| 👥 社交任务 | 50 | 社交活动 |
| 🎯 其他任务 | 50 | 其他类型 |

#### 时长系数

| 预计时长 | 系数 | 说明 |
|---------|------|------|
| 15分钟 | 0.5 | 快速任务 |
| 30分钟 | 0.8 | 短时任务 |
| 1小时 | 1.0 | 标准任务 |
| 2小时 | 1.5 | 中长任务 |
| 4小时 | 2.0 | 长时任务 |
| 8小时 | 3.0 | 超长任务 |

#### 优先级系数

| 优先级 | 系数 | 说明 |
|-------|------|------|
| 1（最高） | 1.5 | 紧急重要 |
| 2（高） | 1.2 | 重要不紧急 |
| 3（中） | 1.0 | 紧急不重要 |
| 4（低） | 0.8 | 不紧急不重要 |

#### 连击系数

| 连续完成 | 系数 | 说明 |
|---------|------|------|
| 1个任务 | 1.0 | 正常 |
| 2个任务 | 1.1 | +10% |
| 3个任务 | 1.2 | +20% |
| 5个任务 | 1.5 | +50% |
| 10个任务 | 2.0 | +100% |

#### 计算示例

**示例 1: 普通工作任务**
```
任务：写工作报告
类型：💼 工作任务（基础 100）
时长：2小时（系数 1.5）
优先级：2（系数 1.2）
连击：第1个（系数 1.0）

金币 = 100 × 1.5 × 1.2 × 1.0 = 180 金币
```

**示例 2: 高优先级学习任务**
```
任务：学习新技能
类型：📚 学习任务（基础 80）
时长：4小时（系数 2.0）
优先级：1（系数 1.5）
连击：第3个（系数 1.2）

金币 = 80 × 2.0 × 1.5 × 1.2 = 288 金币
```

**示例 3: 快速生活任务**
```
任务：整理房间
类型：🏠 生活任务（基础 40）
时长：30分钟（系数 0.8）
优先级：3（系数 1.0）
连击：第1个（系数 1.0）

金币 = 40 × 0.8 × 1.0 × 1.0 = 32 金币
```

## 💡 金币管理技巧

### 1. 合理规划支出

**建议**:
- 💰 保持一定余额（建议 1,000+）
- 🎯 优先兑换实用功能
- 🎁 适度兑换娱乐奖励
- 📊 定期查看收支

**支出比例建议**:
```
功能解锁：40%
虚拟奖励：30%
自定义奖励：20%
储备：10%
```

### 2. 最大化收入

**技巧**:
- ⚡ 优先完成高优先级任务
- 🎯 选择高价值任务类型
- 🔥 利用连击系数
- 📅 每日签到不间断

**收入优化示例**:
```
普通策略：
  3个低优先级任务 × 50金币 = 150金币

优化策略：
  2个高优先级任务 × 150金币 = 300金币
  连击系数 1.1 = 330金币
```

### 3. 设定金币目标

**短期目标**（1周）:
- 赚取 1,000 金币
- 兑换一个主题

**中期目标**（1月）:
- 赚取 5,000 金币
- 解锁高级功能

**长期目标**（3月）:
- 赚取 20,000 金币
- 解锁所有功能

### 4. 避免浪费

**不建议**:
- ❌ 冲动消费
- ❌ 重复购买
- ❌ 低价值兑换
- ❌ 余额过低

**建议**:
- ✅ 计划性消费
- ✅ 优先级排序
- ✅ 高价值兑换
- ✅ 保持储备

## 💎 金币系统总结

### 核心理念

**金币 = 你的努力可视化**

金币系统的目的是：
- 🎯 即时反馈：完成任务立即获得奖励
- 💪 持续激励：积累金币带来成就感
- 🎁 自我奖励：用金币兑换真实奖励
- 📊 量化进步：金币数量反映成长

### 使用原则

1. **赚取为主，消费为辅**
   - 专注完成任务
   - 合理规划支出

2. **追求质量，不只数量**
   - 完成高价值任务
   - 提高任务质量

3. **长期积累，适度奖励**
   - 保持金币储备
   - 适时奖励自己

4. **享受过程，不只结果**
   - 金币是工具，不是目的
   - 成长才是最大的奖励
