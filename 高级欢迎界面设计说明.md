## 🎨 技术实现

### 渐变色方案

```tsx
const gradients = [
  'from-purple-600 via-pink-600 to-red-600',    // 紫粉红 - 激情
  'from-blue-600 via-cyan-600 to-teal-600',     // 蓝青绿 - 科技
  'from-green-600 via-emerald-600 to-teal-600', // 绿翠青 - 成长
  'from-orange-600 via-amber-600 to-yellow-600', // 橙黄 - 活力
];
```

### 动画效果

#### 1. 背景光效
```css
.animate-float-slow {
  animation: float-slow 20s ease-in-out infinite;
}
```
- 缓慢移动的光球
- 营造动态背景
- 增加视觉层次

#### 2. 图标动画
```tsx
const animations = [
  'animate-pulse-slow',    // 脉冲效果
  'animate-bounce-slow',   // 弹跳效果
  'animate-float',         // 浮动效果
  'animate-scale-pulse',   // 缩放脉冲
];
```

#### 3. 内容淡入
```tsx
style={{
  opacity: showContent ? 1 : 0,
  transform: showContent ? 'translateY(0)' : 'translateY(30px)',
  transition: 'all 0.8s cubic-bezier(0.34, 1.56, 0.64, 1) 0.2s',
}}
```
- 从下往上淡入
- 弹性曲线，iOS 风格
- 延迟显示，创造层次

### 进度指示器

```tsx
<div className={`h-1 rounded-full transition-all duration-500 ${
  index === currentStep
    ? 'w-12 bg-white'           // 当前步骤：长条
    : index < currentStep
    ? 'w-8 bg-white/60'         // 已完成：中等透明度
    : 'w-8 bg-white/30'         // 未完成：高透明度
}`} />
```

## 🔧 自定义选项

### 修改文案

在 `WELCOME_STEPS` 数组中修改：

```tsx
{
  id: 1,
  title: '你的标题',
  subtitle: 'Your Subtitle',
  description: '你的描述\n支持换行',
  gradient: 'from-purple-600 via-pink-600 to-red-600',
  icon: <YourIcon className="w-16 h-16" />,
  animation: 'animate-pulse-slow',
}
```

### 修改颜色

```tsx
gradient: 'from-[起始色] via-[中间色] to-[结束色]'
```

### 修改动画

```tsx
animation: 'animate-[动画名称]'
```

可选动画：
- `animate-pulse-slow` - 脉冲
- `animate-bounce-slow` - 弹跳
- `animate-float` - 浮动
- `animate-scale-pulse` - 缩放脉冲

### 添加步骤

在 `WELCOME_STEPS` 数组中添加新对象即可。

## 📊 性能优化

### 1. **CSS 动画**
- 使用 CSS 而非 JS 动画
- GPU 加速（transform, opacity）
- 性能更好，更流畅

### 2. **延迟加载**
- 0.3 秒后才显示内容
- 避免首屏白屏
- 创造悬念效果

### 3. **过渡优化**
- 使用 cubic-bezier 曲线
- 避免线性过渡
- 更自然的动画
