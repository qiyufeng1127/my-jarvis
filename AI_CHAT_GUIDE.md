## 🚀 如何使用

### 步骤 1: 获取 DeepSeek API Key

#### 1.1 访问 DeepSeek 官网

打开浏览器，访问：https://platform.deepseek.com

#### 1.2 注册/登录账号

- 如果没有账号，点击"注册"
- 如果已有账号，直接登录

#### 1.3 获取 API Key

1. 登录后，进入控制台
2. 找到 "API Keys" 或"密钥管理"
3. 点击"创建新密钥"
4. 复制生成的 API Key（格式：`sk-...`）

**重要**: 
- ⚠️ API Key 只显示一次，请妥善保存
- 💰 DeepSeek 提供免费额度，新用户可免费使用

---

### 步骤 2: 配置 API Key

#### 2.1 打开设置

点击顶部导航栏右侧的 **⚙️ 设置** 按钮

#### 2.2 找到 AI 设置

在设置面板中，向下滚动找到 **🤖 AI 设置** 部分

#### 2.3 输入 API Key

1. 在"DeepSeek API Key"输入框中粘贴你的 API Key
2. 点击 **💾 保存 API Key** 按钮
3. 看到"API Key 已保存！"提示

**示例**:
```
DeepSeek API Key
┌─────────────────────────────────┐
│ sk-xxxxxxxxxxxxxxxxxxxxxxxx     │
└─────────────────────────────────┘

[💾 保存 API Key]
```

点击顶部导航栏的 **🤖 AI 助手** 按钮

## ❓ 常见问题

### Q1: 点击 AI 按钮没反应？

**解决方法**:
1. 刷新页面（Ctrl+R 或 F5）
2. 检查浏览器控制台是否有错误
3. 确保已保存代码更改

### Q2: AI 提示"请先配置 API Key"？

**解决方法**:
1. 打开设置 ⚙️
2. 在 AI 设置中输入 API Key
3. 点击保存

### Q3: AI 回复"API 请求失败"？

**可能原因**:
- API Key 不正确
- API Key 已过期
- 网络连接问题
- API 额度用完

**解决方法**:
1. 检查 API Key 是否正确
2. 访问 DeepSeek 官网检查账户状态
3. 检查网络连接
4. 尝试重新生成 API Key

### Q4: AI 回复很慢？

**原因**: 
- 网络延迟
- API 服务器负载

**建议**:
- 耐心等待
- 问题尽量简洁明确

### Q5: 如何获取免费 API Key？

**步骤**:
1. 访问 https://platform.deepseek.com
2. 注册新账号
3. 新用户通常有免费额度
4. 在控制台创建 API Key

## 💡 使用技巧

### 1. 提问技巧

**好的提问**:
- ✅ 具体明确："如何提升执行力？"
- ✅ 包含背景："我经常拖延，有什么方法？"
- ✅ 有明确目标："帮我制定减肥计划"

**不好的提问**:
- ❌ 太模糊："怎么办？"
- ❌ 太宽泛："告诉我所有成长方法"
- ❌ 没有重点："我想变好"

### 2. 充分利用上下文

AI 会记住对话历史，可以连续提问：

```
你: "我想提升执行力"
AI: "可以尝试以下方法..."

你: "第一个方法具体怎么做？"
AI: "关于番茄工作法..."

你: "给我制定一个计划"
AI: "根据你的情况..."
```

### 3. 结合实际数据

提问时可以提供你的实际数据：

```
"我的执行力是82%，专注力73%，应该重点提升哪个？"
"我今天完成了3个任务，还有2个待完成，如何安排？"
"我本周增长了85成长值，这个速度正常吗？"
```

### 4. 寻求具体建议

要求 AI 提供可执行的建议：

```
"给我3个提升专注力的具体方法"
"帮我制定一个30天的成长计划"
"列出今天必须完成的5个任务"
```
