⏰ 智能监控系统 👉 查看监控

工作原理

  1. 任务时间到达 → 自动触发监控
  2. 宽限期倒计时 → 2分钟内完成启动步骤
  3. 预警提醒 → 剩余20秒时提醒
  4. 超时扣币 → 未完成则扣除金币
  5. 循环递增 → 每次超时扣币递增

💡 使用技巧

1. 善用语音助手
开车、做饭、运动时，用语音快速记录："收入200元照相馆客人"
2. 设置合理的财务目标
根据实际情况设置月固定支出、当前欠款，系统会自动计算每日目标
3. 利用金币激励自己
设置你真正想要的奖励：看电影(50币)、买奶茶(30币)、休息一天(200币)
4. 任务拆解
大任务让人焦虑？输入 "帮我拆解：完成项目报告"，AI会自动生成子步骤
5. 记录每一笔收入
养成记录习惯，看着收入增长会很有成就感！

❓ 常见问题

Q: 语音助手没反应？

  • 检查浏览器是否允许麦克风权限
  • 确保使用Chrome或Edge浏览器
  • 尝试点击🎤按钮手动开启 语音设置

Q: 金币不够用怎么办？

  • 按时完成任务获得奖励 查看任务
  • 用人民币兑换金币 去兑换
  • 减少暂停监控的次数

Q: 如何备份/恢复数据？

点击 ⚙️ 设置 → 📤 导出 保存数据 导出数据
点击 ⚙️ 设置 → 📥 导入 恢复数据 导入数据
