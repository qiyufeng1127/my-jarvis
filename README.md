# ADHD Focus - 智能任务管理系统

一个专为ADHD用户设计的智能任务管理系统，集成了AI对话、时间轴、记忆库、游戏化系统等功能。

## 功能特点

- 💬 **智能对话**: 集成DeepSeek AI，提供智能任务建议和对话支持
- 📅 **时间轴**: 可视化任务时间线，帮助规划和追踪任务
- 🧠 **全景记忆库**: 存储和管理重要信息和记忆
- 🎮 **游戏化系统**: 通过金币和能量系统激励任务完成
- 📊 **复盘面板**: 回顾和分析任务完成情况
- 📝 **AI提示词设置**: 自定义AI行为和响应方式
- ✅ **任务验证器**: 验证任务完成情况
- 🚨 **行为矫正系统**: 帮助纠正不良习惯

## 技术栈

- 纯HTML/CSS/JavaScript
- DeepSeek AI API
- 本地存储(LocalStorage)

## 部署

本项目可直接部署到Vercel等静态网站托管平台。

### Vercel部署步骤

1. Fork或导入此仓库到你的GitHub账号
2. 在Vercel中导入项目
3. 部署完成后即可访问

## 使用说明

1. 首次使用需要设置DeepSeek API Key
2. 拖拽组件可以调整位置
3. 点击组件右上角的按钮可以最小化/最大化
4. 使用颜色选择器可以自定义组件背景色

## 获取API Key

访问 [DeepSeek平台](https://platform.deepseek.com/) 获取API Key
