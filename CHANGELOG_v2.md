# 🚀 ADHD Focus v2.0 - AI副驾驶升级

## 核心转变：从"工具"到"AI副驾驶"

> 从"我需要操作它"变成"它主动为我服务"

---

## 🎯 第一阶段：零配置启动

### 每周设置向导
首次启动或每周一自动弹出，AI秘书 KiiKii 会询问：

1. **主要工作类型** - 如：拍照、画插画、写文案
2. **工作时段偏好** - 上午/下午/晚上
3. **收入来源** - 如：摄影工作室、插画约稿
4. **效率挑战** - 如：容易分心、难以开始
5. **新想法** - 本周想尝试的事
6. **家务安排** - 自动分散到一周时间轴

### 自动配置
根据你的回答，系统自动完成：
- 📊 预设任务分类和价值范围
- ⏰ 识别高效时间段
- 💰 根据行业标准预设价值
- 🎯 调整拖延/低效监控参数
- 🏠 智能安排家务到非高效时段

## 🤖 第二阶段：统一AI中枢

### 智能对话作为唯一入口
所有操作都通过自然语言完成，不再需要在不同面板间切换。

**示例对话：**
```
用户：明天画那个插画
AI理解：
  → 查询上下文："那个插画" = 最近提到的"美妆品牌插画"
  → 历史类似任务耗时 = 3小时
  → 用户明天高效时段 = 下午
  
AI回应：
"好的，已安排明天下午2-5点画'美妆品牌插画'，
价值预估900元。需要我拆解步骤吗？"
```

### 上下文感知
- **指代词理解**：说"那个"、"这个"自动关联最近任务
- **时间补全**：说"明天"自动安排到你的高效时段
- **价值估算**：根据任务类型自动估算收入

## 📅 每日主动服务

### 每日问候
每天打开时，KiiKii 会主动告诉你：
- 昨日完成情况
- 今日待办任务
- 预计收入
- 建议的日程安排

### 快捷回复
根据上下文动态变化的快捷按钮：
- ✅ 确认安排
- ✏️ 调整
- 📋 拆解步骤
- ▶️ 开始做

---

## ⏰ 主动提醒系统

### 任务提前提醒
- 任务开始前10分钟弹窗提醒
- 可选择"准备好了"或"推迟10分钟"

### 高效时段提醒
- 进入高效时间段时主动提醒
- 推荐当前最高价值任务
- 一键开始监控

1. 打开页面，查看 KiiKii 的每日问候
2. 确认或调整今日安排
3. 通过对话添加新任务
4. 收到主动提醒时开始工作
5. 享受自动监控和金币奖励
