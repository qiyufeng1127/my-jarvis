# 时间轴导入导出功能说明

## 功能概述

现在你可以在电脑端和手机端之间手动同步时间轴数据了！这个功能让你可以：
- 在电脑上安排工作任务后导出
- 在手机上导入这些任务继续使用
- 反之亦然

## 使用位置

导入/导出按钮位于：
1. **时间轴视图** - 在"时间轴"、"收集箱"、"快捷开始"按钮的右侧
2. **快捷开始视图** - 在页面右上角，设置按钮旁边

## 使用方法

### 📤 导出数据（电脑端）

1. 点击 **"导入/导出"** 按钮
2. 选择 **"导出今日时间轴"**
3. 系统会自动下载一个 JSON 文件，文件名格式：`时间轴_2026-03-06.json`
4. 将这个文件传输到手机（通过微信、QQ、邮件等方式）

### 📥 导入数据（手机端）

1. 在手机上打开应用
2. 点击 **"导入/导出"** 按钮
3. 选择 **"导入时间轴"**
4. 选择之前从电脑导出的 JSON 文件
5. 系统会自动导入所有任务

## 导出内容

导出的数据包含今日的：
- ✅ 所有任务（包括标题、描述、时间等）
- ✅ 任务状态（待办、进行中、已完成）
- ✅ 心情记录
- ✅ 事件记录
- ✅ 日记内容
- ✅ 金币数据（不会修改，按原样导出）
- ✅ 标签和分类
- ✅ 所有任务元数据

## 智能去重

- 导入时会自动检测重复任务（根据任务ID）
- 重复的任务会被自动跳过
- 只有新任务会被导入
- 导入完成后会显示导入数量和跳过数量

## 使用场景

### 场景1：电脑工作 → 手机生活
1. 早上在电脑上安排好一天的工作任务
2. 导出时间轴
3. 在手机上导入
4. 外出时用手机查看和完成任务

### 场景2：手机记录 → 电脑整理
1. 在手机上快速记录临时任务和想法
2. 导出时间轴
3. 在电脑上导入
4. 在电脑上进行详细编辑和规划

### 场景3：数据备份
1. 定期导出时间轴数据
2. 保存 JSON 文件作为备份
3. 需要时可以随时恢复

## 注意事项

⚠️ **重要提示**：
- 导出的是**今日**的所有任务，不包括其他日期
- JSON 文件可以用文本编辑器打开查看
- 建议定期导出备份重要数据
- 导入不会删除现有数据，只会添加新任务
- 文件传输时注意不要修改文件内容

## 技术细节

- 数据格式：JSON
- 版本：1.0
- 包含字段：任务ID、标题、描述、时间、状态、金币、标签等
- 日期格式：ISO 8601 标准
- 文件大小：通常小于 100KB

## 常见问题

**Q: 导出的文件在哪里？**
A: 在浏览器的默认下载文件夹中，文件名包含日期。

**Q: 可以导出多天的数据吗？**
A: 目前只支持导出今日数据。如需导出其他日期，请先切换到对应日期。

**Q: 导入后原有数据会被覆盖吗？**
A: 不会。导入只会添加新任务，不会修改或删除现有数据。

**Q: 可以在不同设备间同步吗？**
A: 可以！只要导出和导入 JSON 文件即可实现手动同步。

**Q: 文件格式错误怎么办？**
A: 确保文件是从应用导出的原始 JSON 文件，没有被修改过。
