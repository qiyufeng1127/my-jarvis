## 🤖 AI 对话功能说明

### ⚠️ 重要说明

**AI 对话功能目前还未完全实现**，但我们有一个语音助手 **Kiki 宝宝** 可以进行基础的语音交互。

---

## 🎤 Kiki 语音助手（当前可用）

### 位置

**右下角的圆形按钮** 🎤

- 粉色圆形按钮
- 固定在屏幕右下角
- 悬停时会放大

### 如何使用

#### 1. 唤醒 Kiki

**操作**:
- 点击右下角的 🎤 按钮

**效果**:
- 按钮变为绿色
- 听到"我在，请说"
- 开始监听你的语音

#### 2. 说出指令

**任务管理**:
```
"创建任务：写工作报告"
"查看今天的任务"
"完成任务"
```

**成长查询**:
```
"我的成长进度"
"查看执行力"
"本周增长了多少"
```

**情感支持**:
```
"给我一点鼓励"
"我很累"
"压力好大"
```

**金币查询**:
```
"查看金币"
"我有多少金币"
```

#### 3. 听 Kiki 回复

Kiki 会：
- 🗣️ 用语音回复你
- 📊 显示相关数据
- 💡 提供建议和鼓励
