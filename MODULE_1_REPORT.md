### 了解度计算公式
```
了解度 = 基础了解度(40%) + 深度了解度(40%) + 时间加成(20%)
```

### 数据结构
```typescript
interface UserProfile {
  usageDays: number;
  understandingLevel: number; // 0-100
  understandingStage: string;
```
