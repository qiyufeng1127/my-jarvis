## 🎯 成长维度系统

### 什么是成长维度？

成长维度是衡量你在不同方面进步的指标。ManifestOS 提供 **5 个核心维度**，全面追踪你的成长。

### 5 个核心维度

#### 1. ⚡ 执行力（红色）

**定义**: 将想法转化为行动的能力

**如何提升**:
- 完成工作任务
- 完成生活任务
- 按时完成计划任务

**影响**:
- 任务完成效率
- 拖延倾向降低
- 行动力增强

**当前值示例**: 82/100

---

#### 2. 🎯 专注力（紫色）

**定义**: 集中注意力完成任务的能力

**如何提升**:
- 完成学习任务
- 完成创作任务
- 长时间专注工作

**影响**:
- 深度工作能力
- 学习效率
- 创造力

**当前值示例**: 73/100

---

#### 3. ❤️ 健康力（绿色）

**定义**: 身体和心理的健康状态

**如何提升**:
- 完成健康任务
- 规律运动
- 良好作息

**影响**:
- 精力水平
- 抗压能力
- 整体状态

**当前值示例**: 65/100

---

#### 4. 💰 财富力（橙色）

**定义**: 创造和管理财富的能力

**如何提升**:
- 完成赚钱相关任务
- 学习理财知识
- 提升专业技能

**影响**:
- 收入水平
- 财务自由度
- 资源掌控力

**当前值示例**: 89/100

---

#### 5. ✨ 魅力值（粉色）

**定义**: 个人吸引力和影响力

**如何提升**:
- 完成社交任务
- 提升形象管理
- 增强沟通能力

**影响**:
- 人际关系
- 社交影响力
- 个人品牌

**当前值示例**: 80/100

## 🏆 身份等级系统

### 等级概述

随着总成长值的增加，你的身份等级会不断提升。每个等级代表不同的成长阶段。

### 5 个身份等级

#### Lv.1 成长探索者
- **成长值范围**: 0 - 500
- **特征**: 刚开始成长之旅
- **解锁功能**: 基础任务管理
- **称号**: 初心者

#### Lv.2 行动实践者
- **成长值范围**: 500 - 1,500
- **特征**: 开始养成行动习惯
- **解锁功能**: 习惯追踪
- **称号**: 实践者

#### Lv.3 习惯建立者
- **成长值范围**: 1,500 - 3,000
- **特征**: 建立稳定的成长习惯
- **解锁功能**: 高级分析
- **称号**: 建立者

#### Lv.4 目标达成者
- **成长值范围**: 3,000 - 5,000
- **特征**: 持续达成重要目标
- **解锁功能**: AI 深度分析
- **称号**: 达成者

#### Lv.5 人生设计师
- **成长值范围**: 5,000+
- **特征**: 掌控人生方向
- **解锁功能**: 全部高级功能
- **称号**: 设计师

### 升级系统

#### 升级进度

**成长进度环**（左侧面板）:
- 圆形进度条
- 显示到下一级的进度百分比
- 实时更新

**计算方式**:
```
当前等级：Lv.1（0-500）
当前成长值：650
下一等级：Lv.2（500-1500）

进度 = (650 - 500) / (1500 - 500) × 100%
     = 150 / 1000 × 100%
     = 15%
```

## 🎯 长期目标系统

### 什么是长期目标？

长期目标是你想要达成的重要人生目标，通过完成相关任务逐步推进。

### 查看长期目标

**位置**: 右侧面板 → 长期目标卡片

**显示内容**:
```

─────────────────────

月入10万
████████████░░░░░░░░ 52%
📈 +3% 本周

健康生活
████████░░░░░░░░░░░░ 40%
📈 +2% 本周

理想身材
████░░░░░░░░░░░░░░░░ 20%
📈 +1% 本周

+ 添加新目标
```

---

### 创建长期目标

#### 步骤 1: 打开创建表单

**操作**:
- 点击长期目标卡片底部的 **"+ 添加新目标"** 按钮

#### 步骤 2: 填写目标信息

**必填字段**:

1. **🎯 目标名称**
   - 简洁明确
   - 示例：`月入10万`、`考取证书`、`减重10斤`

2. **📄 目标描述**
   - 详细说明目标内容
   - 为什么要达成这个目标
   - 达成后的状态

3. **🎨 关联维度**
   - 选择主要关联的成长维度
   - 可以选择多个
   - 示例：月入10万 → 💰 财富力

4. **📅 目标期限**
   - 设定达成目标的时间
   - 示例：`2026年12月31日`

5. **🎯 目标类型**
   - 财富目标
   - 健康目标
   - 学习目标
   - 事业目标
   - 关系目标
   - 其他目标

#### 步骤 3: 创建目标

**操作**:
- 点击 **"创建目标"** 按钮

**效果**:
- 目标出现在目标列表中
- 初始进度为 0%

---

### 更新目标进度

#### 自动更新

**触发条件**:
- 完成关联维度的任务
- 系统自动计算进度

**计算方式**:
```
目标进度 = 关联维度的成长值 / 目标所需成长值 × 100%
```

**示例**:
```
目标：月入10万
关联维度：💰 财富力
当前财富力：89
目标所需：100
进度：89%
```

#### 手动更新

**操作**:
1. 点击目标卡片
2. 在目标详情中调整进度
3. 点击 **"保存"**

**适用场景**:
- 目标进度无法通过任务量化
- 需要根据实际情况调整
- 外部因素影响进度

---

### 目标管理

#### 编辑目标

**操作**:
1. 点击目标卡片
2. 点击 **"编辑"** 按钮
3. 修改目标信息
4. 点击 **"保存"**

#### 完成目标

**操作**:
1. 目标进度达到 100%
2. 系统自动标记为已完成
3. 获得完成奖励

**奖励**:
- 💰 大量金币（根据目标重要性）
- 🏆 成就徽章
- 🎉 庆祝动画

#### 删除目标

**操作**:
1. 点击目标卡片
2. 点击 **"删除"** 按钮
3. 确认删除

**注意**:
- 删除操作不可恢复
- 已完成的目标建议归档而非删除

## 💡 成长系统使用技巧

### 1. 平衡发展

**建议**:
- 不要只关注单一维度
- 保持各维度相对平衡
- 薄弱维度重点提升

**检查方式**:
```
理想状态：各维度差距不超过 20%

⚡ 执行力：82%  ✅
🎯 专注力：73%  ✅
❤️ 健康力：65%  ⚠️ 需要提升
💰 财富力：89%  ✅
✨ 魅力值：80%  ✅
```

### 2. 设定合理目标

**SMART 原则**:
- **S**pecific（具体的）
- **M**easurable（可衡量的）
- **A**chievable（可达成的）
- **R**elevant（相关的）
- **T**ime-bound（有时限的）

**示例**:
```
❌ 不好：变得更好
✅ 好：3个月内减重10斤，体脂率降到20%
```

### 3. 定期回顾

**建议频率**:
- 每日：查看当日成长
- 每周：回顾周报，调整计划
- 每月：深度分析，设定新目标

### 4. 关联任务和目标

**操作**:
- 创建任务时选择关联目标
- 确保任务推进目标进度
- 避免无效任务

**示例**:
```
目标：月入10万
关联任务：
  ✅ 学习新技能（提升能力）
  ✅ 完成项目（增加收入）
  ✅ 拓展人脉（创造机会）
```
