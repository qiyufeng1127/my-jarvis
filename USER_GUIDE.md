## 🎯 系统概述

ADHD Focus 是一个**专为ADHD人群设计**的智能任务管理系统。它不仅仅是一个待办清单，而是一个**AI驱动的效率伙伴**，通过以下核心理念帮助你：

- 🎮 **游戏化激励** - 完成任务获得金币，让工作像游戏一样有趣
- 💰 **价值可视化** - 实时看到你赚了多少钱，激发动力
- 🤖 **AI智能助手** - KiiKii 学习你的习惯，越用越懂你
- ⏰ **拖延克星** - 自动监控任务启动，用金币惩罚推动行动
- 🧠 **记忆系统** - 记录你的情绪、想法、习惯，提供个性化建议

## 🚀 快速开始（5分钟上手）

### 🌟 首次访问

首次打开系统时，你会看到一个**3步引导教程**：

1. **欢迎页** - 了解系统核心功能
2. **智能对话介绍** - 学习如何与KiiKii交流
3. **快捷键提示** - 掌握高效操作方式

> 💡 **提示**：如果跳过了引导，可以在设置面板中点击"重新引导"再次查看

### 第一步：设置API Key（启用AI功能）

1. 点击智能对话框右上角的 🔑 按钮
2. 输入你的 DeepSeek API Key
3. 点击"保存"
4. 看到绿色状态点和"🤖 KiiKii 在线"提示即表示连接成功

> 🔗 **获取API Key**: 访问 [platform.deepseek.com](https://platform.deepseek.com/)

### 第二步：设置财务目标

系统会自动弹出财务设置窗口，输入：

| 设置项 | 说明 | 示例 |
|--------|------|------|
| 💳 当前欠款 | 信用卡、花呗、借款等 | ¥8,500 |
| 🏠 每月固定支出 | 房租、水电、生活费等 | ¥6,300 |
| 🎯 每月收入目标 | 可选，用于计算进度 | ¥15,000 |

点击"开始赚钱！"完成设置。

### 第三步：添加第一个任务

在智能对话框输入，例如：
```
明天下午3点开会，预计1小时
```

KiiKii会自动：
- ✅ 解析时间（明天15:00）
- ✅ 创建任务卡片
- ✅ 添加到时间轴
- ✅ 估算任务价值

## 💾 数据备份与恢复（v2.0新增）

### 导出数据

1. 点击左下角主题按钮旁边区域，或按 `Ctrl + ,` 打开设置
2. 点击"📤 导出"按钮
3. 系统会下载一个 `adhd-focus-backup-日期.json` 文件

**导出内容包括：**
- 所有任务数据
- 情绪记忆记录
- 游戏状态（金币、等级、成就）
- 用户配置
- AI记忆数据
- AI学习数据
- 财务设置
- 主题偏好

1. 打开设置面板
2. 点击"📥 导入"按钮
3. 选择之前导出的 `.json` 备份文件
4. 确认后页面会自动刷新，数据恢复完成

> ⚠️ **注意**：导入会覆盖当前所有数据，请谨慎操作

## ⏰ 拖延监控详解

### 什么是拖延监控？

拖延监控专门针对**任务启动阶段**——这是ADHD人群最容易卡住的环节。当任务开始时间到达时，系统会自动触发监控，用**倒计时+金币惩罚**推动你立即行动。

### 工作流程

```
任务开始时间到达
        ↓
  自动触发拖延监控
        ↓
  显示启动步骤（如"打开电脑"）
        ↓
  开始倒计时（默认2分钟）
        ↓
  ┌─────────────────────────────────┐
  │                                 │
  ↓                                 ↓
[按时完成]                      [超时未完成]
  ↓                                 ↓
获得3金币奖励 ✅                 扣除金币 💸
  ↓                                 ↓
进入下一步骤                    重新倒计时
                                    ↓
                              循环直到完成
```

### 界面说明

```
┌─────────────────────────────────────┐
│ ⏰ 拖延监控                  [_][□] │
├─────────────────────────────────────┤
│  ┌─────────────────────────────┐    │
│  │ 🎨 画插画                   │    │
│  │                             │    │
│  │ 💡 启动步骤: 打开PS软件     │    │
│  │                             │    │
│  │      ⏱️ 01:45              │    │
│  │                             │    │
│  │ [倒计时中] [第1轮] [已扣0🪙]│    │
│  │                             │    │
│  │ [✅ 已完成启动步骤]         │    │
│  └─────────────────────────────┘    │
│                                     │
│ 📜 历史记录                         │
│ ├─ 09:15 写报告 ✅ 按时启动 +3🪙   │
│ ├─ 08:30 晨跑 💸 超时1次 -5🪙      │
│ └─ ...                              │
│                                     │
│ ⚙️ 设置                             │
│ 宽限期: [2分钟 ▼]                   │
│ 预警时间: [20秒 ▼]                  │
└─────────────────────────────────────┘
```

### 倒计时状态

| 颜色 | 状态 | 说明 |
|------|------|------|
| 🟢 绿色 | 正常 | 时间充裕 |
| 🟡 黄色 | 警告 | 剩余不到30秒 |
| 🔴 红色 | 紧急 | 剩余不到10秒，闪烁提醒 |

### 金币规则

| 情况 | 金币变化 |
|------|----------|
| 按时完成启动步骤 | +3 🪙 |
| 第1次超时 | -5 🪙 |
| 第2次超时 | -7 🪙 (+50%) |
| 第3次超时 | -11 🪙 |
| 最高上限 | -50 🪙 |

### 设置选项

| 设置项 | 可选值 | 默认 | 说明 |
|--------|--------|------|------|
| 启动宽限期 | 1/2/3/5分钟 | 2分钟 | 完成启动步骤的时间限制 |
| 预警提前时间 | 10/20/30秒 | 20秒 | 倒计时结束前多久提醒 |
| 预警提示语 | 自定义 | "还有{seconds}秒！" | 支持变量替换 |

## 📉 低效率监控详解

### 什么是低效率监控？

低效率监控针对**任务执行阶段**——当你开始做任务后，可能会陷入卡顿、分心、或者在某个步骤上花费过长时间。系统会监控你的执行效率，及时提供帮助。

### 工作流程

```
选择任务开始监控
        ↓
  显示当前执行步骤
        ↓
  开始计时（默认1小时判定）
        ↓
  半程提醒（30分钟时）
  "一切顺利吗？"
        ↓
  ┌─────────────────────────────────┐
  │                                 │
  ↓                                 ↓
[按时完成步骤]                  [超时未完成]
  ↓                                 ↓
获得2金币 ✅                    显示警报 ⚠️
  ↓                                 ↓
进入下一步骤                    显示辅助选项
                                    ↓
                              选择处理方式
```

### 辅助选项

当执行超时时，系统提供4种辅助方式：

| 选项 | 图标 | 说明 |
|------|------|------|
| 支付金币重置 | 🪙 | 花费金币重新开始计时 |
| AI微调步骤 | 🤖 | 让AI把当前步骤拆得更小 |
| 正念呼吸 | 🧘 | 5分钟呼吸练习，重置状态 |
| 切换任务 | 🔄 | 暂时切换到轻松任务 |

### 效率评分

系统会根据你的执行情况计算效率分数：

| 评分 | 颜色 | 说明 |
|------|------|------|
| 90-100% | 🟢 绿色 | 优秀，保持！ |
| 70-89% | 🟡 黄色 | 良好，可以更好 |
| 50-69% | 🟠 橙色 | 一般，需要注意 |
| <50% | 🔴 红色 | 较低，建议调整 |

## ❓ 常见问题FAQ

### 基础问题

**Q: 任务没有自动触发拖延监控？**

A: 请检查：
1. 任务是否添加到**今天**的时间轴
2. 任务开始时间是否**已到达**
3. 拖延监控面板是否显示"已启用"

**Q: AI功能不工作？**

A: 请确认：
1. 已正确设置API Key（点击🔑按钮）
2. 网络连接正常
3. API余额充足
4. 状态点显示绿色

**Q: 如何修改财务目标？**

A: 点击价值显化器右上角的 ⚙️ 按钮，重新设置欠款和支出。

**Q: 金币不够扣怎么办？**

A: 系统会扣除你现有的全部金币，不会变成负数。但这意味着你需要更努力赚回来！

### v2.0新功能问题

**Q: 如何切换深色模式？**

A: 三种方式：
1. 点击左下角 🌙 按钮
2. 按快捷键 `Ctrl + D`
3. 打开设置面板切换开关
