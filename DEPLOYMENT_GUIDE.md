# 部署后清除缓存指南

## 问题：更新后看不到新功能

由于这是 PWA 应用，浏览器会缓存资源。更新后需要清除缓存。

## 解决方案

### 手机端（推荐）

1. **强制刷新页面**
   - iOS Safari: 长按刷新按钮 → 选择"清除缓存并刷新"
   - Android Chrome: 设置 → 隐私 → 清除浏览数据 → 选择"缓存的图片和文件"

2. **卸载并重新安装 PWA**
   - 从主屏幕删除应用图标
   - 重新访问网站
   - 重新"添加到主屏幕"

3. **使用隐私模式测试**
   - 在隐私/无痕模式下打开网站
   - 这样可以看到最新版本

### 电脑端

1. **Chrome/Edge**
   - 按 `Ctrl + Shift + Delete` (Windows) 或 `Cmd + Shift + Delete` (Mac)
   - 选择"缓存的图片和文件"
   - 点击"清除数据"
   - 刷新页面 `Ctrl + F5`

2. **开发者工具强制刷新**
   - 按 `F12` 打开开发者工具
   - 右键点击刷新按钮
   - 选择"清空缓存并硬性重新加载"

## 验证更新是否生效

### 检查清单：

- [ ] 手机端点击"设置"，页面不再白屏，能看到内容
- [ ] 长按底部导航栏进入编辑模式，能看到🎨颜色按钮
- [ ] 左下角的"语音指令帮助"按钮已消失
- [ ] 点击"更多"或"设置"时，右下角的AI和语音按钮自动隐藏
- [ ] 设置中的"云同步"标签页能看到"生成同步码"和"加入同步码"功能

## 如果还是不行

1. 检查 Vercel 部署状态：https://vercel.com/dashboard
2. 确认最新提交已部署：commit `1c02c05`
3. 查看浏览器控制台是否有错误信息
