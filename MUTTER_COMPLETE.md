## 📊 功能特性

### 1. 智能识别
- 自动识别口语化表达（"啊啊啊"、"糟了"、"好烦"）
- 识别情绪宣泄和吐槽
- 识别生活琐事和随意想法

### 2. 心情分析
- AI自动分析心情状态
- 生成对应的心情emoji
- 智能打上情绪标签

### 3. 彩色卡片
根据情绪自动选择卡片颜色：
- 😊 开心/兴奋 → 金色
- 😣 难过/沮丧 → 浅蓝灰
- 😤 生气/烦躁 → 红色
- 😰 焦虑/不适 → 薰衣草紫
- 😴 疲惫 → 浅灰
- 😌 平静/感恩 → 薄荷绿
- 💭 默认 → 粉红色

### 4. 真人对话
AI会根据性格设置给出个性化回复：
- **毒舌教练**：直接犀利，不留情面
- **温柔体贴**：温暖关心，像闺蜜
- **专业顾问**：正式严谨，注重效率
- **好朋友**：轻松随意，偶尔调侃

### 5. 记忆保存
- 所有碎碎念保存到记忆库
- 可以在全景记忆栏查看
- 支持按标签筛选和搜索

## 🔧 调整AI性格

如果想调整AI的回复风格：

1. 点击AI助手左上角的头像
2. 选择预设性格或自定义调整
3. 调整"毒舌程度"滑块：
   - 0-30：温柔体贴
   - 40-70：中等友好
   - 70-100：毒舌犀利
4. 保存设置

## 🐛 故障排除

### 如果AI没有回复
1. 检查是否配置了DeepSeek API Key
2. 打开浏览器控制台（F12）查看错误信息
3. 如果AI不可用，会使用降级回复

### 如果卡片没有创建
1. 检查浏览器控制台是否有错误
2. 确认时间轴模块正常工作
3. 刷新页面重试

### 如果识别不准确
1. 尝试更口语化的表达
2. 使用更多情绪词汇（"啊啊啊"、"好烦"、"累死了"）
3. AI会持续学习和改进

## 📝 使用建议

1. **随时记录**：有任何想法、情绪、吐槽都可以直接输入
2. **真实表达**：不用担心措辞，越真实越好
3. **查看回顾**：定期查看时间轴上的碎碎念卡片，了解自己的情绪变化
4. **调整性格**：根据自己的喜好调整AI的性格，找到最舒服的对话方式
