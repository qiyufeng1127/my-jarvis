# 📅 时间轴日历组件使用指南

## 🎯 设计理念

**"时间是有形的容器"** - 将抽象的时间转化为可触摸、可移动的块，让你直面一天的有限性。

---

## ✨ 核心特性

### 1. 可视化时间块 ⏰

**特点**:
- 每个任务显示为彩色块
- 长度代表持续时间
- 直观感受时间消耗

**优势**:
- 避免安排过多任务
- 准确估计时间需求
- 培养时间感知能力

---

### 2. 拖拽调整 🎯

**功能**:
- 拖动任务块改变开始时间
- 拖动边缘调整时长
- 像排列积木一样简单

**操作**:
- 点击并拖动任务块
- 松开鼠标完成调整
- 实时预览新位置

---

### 3. 颜色编码 🎨

**任务类别颜色**:
- 💼 工作任务：粉色 (#DD617C)
- 📚 学习任务：绿色 (#6D9978)
- 💪 健康任务：黄色 (#E8C259)
- 🏠 生活任务：米色 (#D1CBBA)
- 👥 社交任务：红色 (#AC0327)
- 🎯 其他任务：灰色 (#9CA3AF)

**自定义**:
- 可以为每个任务设置颜色
- 快速区分不同类别
- 视觉上更清晰

### 4. 三种视图模式 📊

#### 日视图（时间轴）

**特点**:
- 24小时垂直时间轴
- 精确到分钟的规划
- 当前时间红线指示

**适用场景**:
- 详细的日程规划
- 精确的时间管理
- 执行当天任务

**界面布局**:
```
┌─────────────────────────────────────┐
│ ← 1月22日 周三 →  [今天]  日周月   │
├─────────────────────────────────────┤
│ 全天任务：[任务1] [任务2]          │
├──────┬──────────────────────────────┤
│ 00:00│                              │
│ 01:00│                              │
│ 02:00│                              │
│  ...  │  [任务块]                    │
│ 09:00│  ┌──────────────┐            │
│ 10:00│  │ 写工作报告   │            │
│ 11:00│  │ 2小时        │            │
│ 12:00│  └──────────────┘            │
│  ...  │  ← 当前时间线                │
│ 23:00│                              │
└──────┴──────────────────────────────┘
```

#### 周视图

**特点**:
- 7天横向排列
- 显示每天的任务概览
- 快速对比一周安排

**适用场景**:
- 周计划制定
- 工作量平衡
- 查看任务分布

**界面布局**:
```
┌─────────────────────────────────────┐
│ ← 1月19日 - 1月25日 →               │
├──────┬──────┬──────┬──────┬────────┤
│  日  │  一  │  二  │  三  │  ...   │
│  19  │  20  │  21  │  22  │        │
├──────┼──────┼──────┼──────┼────────┤
│[任务]│[任务]│[任务]│[任务]│        │
│[任务]│      │[任务]│[任务]│        │
│      │      │      │[任务]│        │
└──────┴──────┴──────┴──────┴────────┘
```

#### 月视图

**特点**:
- 传统日历网格
- 任务密度指示点
- 快速跳转日期

**适用场景**:
- 月度规划
- 查看整体安排
- 选择特定日期

**界面布局**:
```
┌─────────────────────────────────────┐
│ ← 2026年1月 →                       │
├──────┬──────┬──────┬──────┬────────┤
│  日  │  一  │  二  │  三  │  ...   │
├──────┼──────┼──────┼──────┼────────┤
│      │      │   1  │   2  │        │

├──────┼──────┼──────┼──────┼────────┤
│   5  │   6  │   7  │   8  │        │
```

### 5. 全天任务 vs 时间段任务 📌

#### 全天任务

**特点**:
- 显示在时间轴顶部
- 不占用具体时间段
- 横向排列

**适用场景**:
- 不需要固定时间的任务
- 灵活安排的事项
- 提醒类任务

**示例**:
```
全天任务：
[整理房间] [购物] [回复邮件]
```

#### 时间段任务

**特点**:
- 显示在时间轴中
- 占用具体时间段
- 垂直排列

**适用场景**:
- 需要专注时间的任务
- 有明确时间要求的事项
- 会议、约会等

**示例**:
```
09:00 ┌──────────────┐
      │ 团队会议     │
10:00 └──────────────┘
      
14:00 ┌──────────────┐
      │ 写代码       │
      │              │
17:00 └──────────────┘
```

## 🎮 交互操作

### 基础操作

#### 1. 创建任务

**方法 1: 点击按钮**
- 点击右上角 **"+ 新建任务"** 按钮
- 填写任务信息
- 选择开始时间和时长

**方法 2: 下拉创建（计划中）**
- 在时间轴上向下拉
- 自动创建新任务
- 快速设置时间

#### 2. 查看任务详情

**操作**:
- 点击任务块
- 查看完整信息
- 可以编辑或删除

#### 3. 标记完成

**操作**:
- 点击任务块
- 任务变灰并显示 ✓
- 自动记录完成时间

#### 4. 删除任务

**方法 1: 点击删除按钮**
- 鼠标悬停在任务块上
- 点击右上角垃圾桶图标
- 确认删除

**方法 2: 左滑删除（计划中）**
- 在任务块上向左滑动
- 显示删除按钮
- 点击确认

---

### 高级操作

#### 1. 拖拽调整时间

**操作**:
1. 点击并按住任务块
2. 拖动到新的时间位置
3. 松开鼠标完成调整

**效果**:
- 实时预览新位置
- 自动对齐时间刻度
- 避免任务重叠

#### 2. 调整任务时长

**操作**:
1. 鼠标悬停在任务块边缘
2. 光标变为调整大小图标
3. 拖动边缘改变时长

**效果**:
- 动态调整任务块高度
- 实时显示新时长
- 自动计算结束时间

#### 3. 缩放时间轴

**操作**:
- 点击 **"放大"** 按钮：更精细的时间刻度
- 点击 **"缩小"** 按钮：更粗放的时间刻度

**效果**:
- 15分钟/格 → 精细规划
- 30分钟/格 → 标准视图
- 60分钟/格 → 概览视图
- 120分钟/格 → 粗放规划

**快捷键（计划中）**:
- 捏合手势：缩放时间轴
- Ctrl + 滚轮：缩放

#### 4. 快速跳转日期

**操作**:
- 点击 **"← →"** 按钮：前后一天/周/月
- 点击 **"今天"** 按钮：回到今天
- 切换到月视图：点击任意日期跳转

## 🎨 视觉反馈

### 1. 任务状态

**未完成**:
- 彩色背景
- 正常透明度
- 可以拖拽

**已完成**:
- 灰色背景
- 降低透明度

**特点**:
- 红色横线
- 左侧圆点

**作用**:
- 追踪当前进度

- 提醒时间流逝

### 3. 任务密度

## 💡 使用技巧

### 1. 合理规划时间

**建议**:
- 每天不超过8小时的任务
- 留出缓冲时间
- 考虑休息和吃饭

**示例**:
```
09:00-10:00  任务1 (1小时)
10:00-10:15  休息 (15分钟)
10:15-12:00  任务2 (1.75小时)
12:00-13:00  午餐 (1小时)
13:00-15:00  任务3 (2小时)
15:00-15:15  休息 (15分钟)
15:15-17:00  任务4 (1.75小时)
```

### 2. 颜色分类

**建议**:
- 工作任务用粉色
- 学习任务用绿色
- 健康任务用黄色
- 生活任务用米色

**效果**:
- 一眼看出任务类型
- 平衡不同类别
- 避免单一类型过多

### 3. 时间块大小

**建议**:
- 最小任务：15分钟
- 标准任务：30分钟-2小时
- 大任务：拆分成多个小任务

**原因**:
- 太小的任务难以管理
- 太大的任务容易拖延
- 适中的任务最高效

### 4. 使用全天任务

**适合全天任务**:
- 不需要固定时间
- 可以随时完成
- 灵活安排

**示例**:
- 整理房间
- 购物
- 回复邮件
- 打电话

**不适合全天任务**:
- 需要专注时间
- 有明确时间要求
- 会议、约会

### 5. 定期回顾

**每日回顾**:
- 查看完成情况
- 调整明天计划
- 总结经验教训

**每周回顾**:
- 查看周视图
- 分析时间分配
- 优化下周安排

**每月回顾**:
- 查看月视图
- 评估目标进度
- 制定下月计划

## 🚀 工作流程

### 理想的一天

#### 早上（规划）

1. **打开日视图**
   - 查看今天的任务
   - 检查时间安排

2. **调整计划**
   - 拖拽任务调整时间
   - 添加新任务
   - 删除不必要的任务

3. **确认优先级**
   - 重要任务放在精力最好的时段
   - 简单任务放在精力较低的时段

#### 白天（执行）

1. **跟随时间线**
   - 看当前时间指示线
   - 执行对应的任务

2. **标记完成**
   - 完成任务后点击标记
   - 任务变灰显示 ✓

3. **灵活调整**
   - 遇到突发情况拖拽调整
   - 实时更新计划

#### 晚上（回顾）

1. **查看完成情况**
   - 统计完成的任务
   - 分析未完成的原因

2. **规划明天**
   - 切换到明天
   - 添加明天的任务

3. **总结经验**
   - 记录时间估计是否准确
   - 调整未来的规划策略

## 🎯 设计优势

### 1. 降低规划阻力

**传统方式**:
- 在待办列表中焦虑
- 不知道何时做
- 容易拖延

**时间轴方式**:
- 拖拽即可安排
- 清晰看到时间
- 鼓励提前规划

### 2. 提高时间感知

**视觉化**:
- 看到任务消耗的"面积"
- 直观感受时间有限性
- 培养准确的时间感

**效果**:
- 更准确估计时间
- 避免安排过多
- 提高完成率

### 3. 统一工作流

**三合一**:
- 规划（何时做）
- 执行（看到当前任务）
- 记录（实际用时）

**优势**:
- 减少应用间切换
- 流程更顺畅
- 效率更高

## ✅ 快速开始

### 步骤 1: 打开日历

点击顶部导航栏的 **📅 日历** 按钮

- 拖拽任务块调整时间
- 拖拽边缘调整时长
- 删除不需要的任务

### 步骤 5: 执行任务

- 跟随当前时间线
- 完成后点击标记
- 实时调整计划
