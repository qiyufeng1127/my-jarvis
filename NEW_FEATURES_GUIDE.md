## 🔊 语音警报功能

### 拖延监控语音提醒

**功能说明：**
- 任务到时间时，系统会循环播放语音提醒
- 支持自定义提示语内容
- 可以用金币暂停30分钟

**使用步骤：**

1. **启用语音提醒**
   - 打开拖延监控面板
   - 确保"使用语音播报"已开启
   - 首次使用需要点击任意按钮激活语音权限

2. **自定义提示语**（可选）
   - 在设置中找到"拖延监控设置"
   - 修改"自定义预警提示语"和"自定义超时提示语"
   - 可使用变量：`{task}` `{step}` `{seconds}`

3. **使用暂停功能**
   - 当语音警报响起时
   - 点击"💰 暂停提醒"按钮
   - 支付10金币即可暂停30分钟
   - 30分钟后自动恢复提醒

### 低效率监控语音提醒

**功能说明：**
- 在步骤停留过久时循环播放语音提醒
- 帮助您及时发现低效状态
- 同样支持金币暂停功能

**使用步骤：**

1. **开始监控任务**
   - 在低效率监控面板中选择任务
   - 点击"开始监控"

2. **接收语音提醒**
   - 超过判定时长后，系统会循环播放语音
   - 提醒您可能陷入低效循环

3. **暂停或完成**
   - 点击"💰 暂停提醒"暂停30分钟
   - 或点击"✅ 完成步骤"进入下一步

## 🤖 Kiki 智能问候

### 优化说明

Kiki 现在更聪明了！不会频繁打扰您，只在以下情况发送问候：

- ✅ 新的一天开始，且有待办任务
- ✅ 周一（每周开始）
- ✅ 有重要通知需要提醒
- ❌ 不再频繁显示"下午好"等常规问候

### 效果

- 减少干扰，提高专注度
- 只在真正需要时才提醒
- 让 Kiki 成为您的智能助手，而非打扰者

## ⚙️ 详细设置

### 拖延监控设置

**位置：** 拖延监控面板 → 设置按钮

**可配置项：**
- 启动宽限期（默认2分钟）
- 预警提前时间（默认20秒）
- 自定义预警提示语
- 自定义超时提示语
- 语音循环间隔（默认10秒）
- 金币暂停成本（默认10金币）

**推荐设置：**
```
启动宽限期：120秒（2分钟）
预警提前时间：20秒
语音循环间隔：10秒
自定义提示语：距离{task}开始还有{seconds}秒，请准备！
```

### 低效率监控设置

**位置：** 低效率监控面板 → 设置按钮

**可配置项：**
- 判定时长（默认60分钟）
- 半程提醒（建议开启）
- 自定义警报提示语
- 语音循环间隔（默认15秒）
- 金币暂停成本（默认10金币）

**推荐设置：**
```
判定时长：60分钟
半程提醒：开启
语音循环间隔：15秒
自定义提示语：已停留{minutes}分钟，需要调整吗？
```

## 🔧 故障排除

### 语音不播放？

1. **检查浏览器权限**
   - 确保允许网站播放声音
   - 某些浏览器需要手动授权

2. **激活语音功能**
   - 点击任意按钮（浏览器安全限制）
   - 刷新页面后重试

3. **检查设置**
   - 确认"使用语音播报"已开启
   - 检查系统音量是否静音

### 自定义导航不保存？

1. **检查浏览器存储**
   - 确保未开启无痕模式
   - 检查浏览器存储权限

2. **清除缓存后重试**
   - 导出数据备份
   - 清除浏览器缓存
   - 重新配置

### 暂停功能无效？

1. **检查金币余额**
   - 确保有足够的金币（至少10个）
   - 完成任务获取更多金币

2. **检查暂停状态**
   - 查看是否已在暂停中
   - 等待暂停结束后重试
