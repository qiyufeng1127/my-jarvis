# AI总结提示词优化 - 让小票更有温度

## 🎯 优化目标

让AI生成的每日总结：
- 💬 **口语化**: 像朋友聊天一样自然
- ❤️ **有温度**: 该夸就走心夸，该批评就毒舌点，该安慰就温柔点
- 📊 **基于数据**: 根据真实数据分析，不乱说
- 💡 **醍醐灌顶**: 让人有感触，有收获
- 🎨 **每次不同**: 避免重复，保持新鲜感

## 📝 提示词设计

### 核心理念
把AI当成一个**真诚的朋友**，而不是冷冰冰的机器人。朋友会：
- 看到你努力时，真心为你高兴
- 看到你偷懒时，毒舌但关心地提醒
- 看到你低谷时，温柔地安慰和鼓励
- 给出具体的建议，而不是空话套话

### 提示词结构

#### 1. 角色定位
```
你是用户的贴心朋友，正在看他今天的任务完成情况。
请用真诚、口语化的方式给出一段走心的总结。
```

#### 2. 数据输入
```
【今日数据】
- 完成任务：X/Y（完成率Z%）
- 获得金币：XXX
- 效率分数：XX/100
- 任务列表：任务1(✓)、任务2(×)、任务3(✓)...
```

#### 3. 语气要求
```
- 像朋友聊天一样自然、口语化
- 该夸就走心地夸，该批评就毒舌一点，该安慰就温柔一点
- 不要客套话，要真诚、直接、有温度
```

#### 4. 内容要求
```
- 基于真实数据分析，不要乱说
- 指出具体的亮点或问题
- 给出实用的建议或鼓励
- 让人有醍醐灌顶的感觉
```

#### 5. 风格分级

**完成率90%+（优秀）**:
```
走心夸奖，比如：
"今天真的太棒了！XX任务完成得特别漂亮，这种状态保持下去，
你就是自己的人生赢家！"
```

**完成率60-90%（良好）**:
```
中肯评价，比如：
"今天表现不错，但XX任务没完成有点可惜。
明天早点开始，你肯定能做得更好！"
```

**完成率30-60%（一般）**:
```
温柔提醒，比如：
"今天有点拖延哦，XX任务都没动。
是不是遇到什么困难了？明天咱们调整一下节奏试试？"
```

**完成率<30%（较差）**:
```
毒舌但关心，比如：
"说好的要努力呢？今天基本在摸鱼啊！
不过没关系，谁还没有低谷期呢，明天重新开始！"
```

#### 6. 格式要求
```
- 50字以内，简洁有力
- 必须包含emoji增加趣味性
- 每次生成的文案都要不一样，避免重复
```

## 🔧 技术实现

### 关键参数

```typescript
// 提高temperature，增加创意和多样性
temperature: 0.9  // 从0.8提高到0.9

// System prompt强调多样性
role: 'system',
content: '你是一个真诚、有温度的朋友，擅长用口语化的方式给出走心的反馈。
         你会根据数据给出客观评价，该夸就夸，该批评就批评，
         但始终保持关心和鼓励的态度。
         每次生成的文案都要不一样，避免重复。'
```

### 降级方案

如果AI调用失败，根据完成率返回不同的默认文案：

```typescript
if (completionRate >= 90) {
  return `今天效率爆表！完成${completed}个任务，${gold}金币到手💰 
          这种状态简直无敌，明天继续冲鸭！`;
} else if (completionRate >= 70) {
  return `完成${completed}个任务，还不错👍 
          不过还有${total - completed}个没做完，
          明天早点开始，你肯定能全部搞定！`;
} else if (completionRate >= 40) {
  return `今天有点摸鱼哦😅 ${total - completed}个任务没动，
          是不是遇到困难了？明天调整节奏，咱们重新来过！`;
} else {
  return `说实话，今天基本在划水😑 ${completed}/${total}的完成率有点说不过去啊！
          明天能不能给力点？加油！💪`;
}
```

## 💡 设计亮点

### 1. 真实数据驱动
- ✅ 基于实际完成任务数
- ✅ 基于实际获得金币数
- ✅ 基于实际效率分数
- ✅ 基于具体任务列表

### 2. 情感分级
- 😊 优秀：走心夸奖，让人有成就感
- 👍 良好：中肯评价，指出改进方向
- 😅 一般：温柔提醒，给予鼓励
- 😑 较差：毒舌但关心，激发斗志

### 3. 口语化表达
- ❌ 不说："您今日的任务完成情况良好"
- ✅ 要说："今天表现不错，继续加油！"

- ❌ 不说："建议您提高执行效率"
- ✅ 要说："明天早点开始，你肯定能做得更好！"

### 4. 具体化建议
- ❌ 不说："继续努力"
- ✅ 要说："那几个没完成的任务，明天优先处理掉"

- ❌ 不说："需要改进"
- ✅ 要说："'学习'那个任务拖了点，明天集中精力搞定它"

### 5. 多样性保证
- 🎲 高temperature（0.9）增加随机性
- 🔄 System prompt强调"每次都要不一样"
- 📝 提供多种风格示例
- 🎨 鼓励创意表达

## 🎯 预期效果

### 用户感受
1. **有温度**: "这不是机器人，这是我的朋友在关心我"
2. **有收获**: "说得对，我确实该改进这个地方"
3. **有动力**: "明天我要做得更好！"
4. **有趣味**: "每天的总结都不一样，很期待"

### 数据驱动
- 完成率高 → 真心夸奖 → 增强成就感 → 继续保持
- 完成率中 → 中肯评价 → 明确方向 → 努力改进
- 完成率低 → 温柔提醒 → 给予鼓励 → 重新出发

### 长期价值
- 📈 提高用户粘性（每天都想看总结）
- 💪 增强执行动力（被朋友鼓励的感觉）
- 🎯 明确改进方向（具体的建议）
- ❤️ 建立情感连接（有温度的产品）
