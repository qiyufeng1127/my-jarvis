## 🐛 常见问题排查

### 问题 1: 图表不显示

**原因**: Chart.js 依赖包未安装

**解决方案**:
```bash
npm install chart.js react-chartjs-2
```

---

### 问题 2: 语音识别不工作

**原因**: 浏览器未授予麦克风权限

**解决方案**:
1. 点击浏览器地址栏左侧的锁图标
2. 允许麦克风权限
3. 刷新页面

---

### 问题 3: 模块拖拽不流畅

**原因**: 浏览器性能问题

**解决方案**:
1. 关闭其他标签页
2. 使用 Chrome 或 Edge 浏览器
3. 减少同时显示的模块数量

---

### 问题 4: AI 功能不工作

**原因**: DeepSeek API 未配置

**解决方案**:
1. 申请 DeepSeek API Key
2. 在 `.env` 文件中配置
3. 重启开发服务器

---

### 问题 5: 数据刷新后丢失

**原因**: 数据未持久化到数据库

**解决方案**:
- 这是预期行为，数据持久化功能待实现
- 当前所有数据存储在内存中
- 刷新页面会重置数据
