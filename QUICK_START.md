# ADHD Focus 快速入门卡片

## 🚀 30秒快速开始

```
1️⃣ 设置财务目标 → 点击价值显化器的⚙️按钮
2️⃣ 输入任务 → 在智能对话框说"明天3点拍照，2小时"
3️⃣ 开始干活 → 任务时间到自动触发监控
4️⃣ 完成赚钱 → 每完成一步，叮咚！收入+1
```

---

## 📝 智能对话输入示例

| 你说的话 | 系统理解 |
|---------|---------|
| "明天下午3点拍婚纱照" | 📅 明天15:00 📌 拍婚纱照 |
| "画插画800元，3小时" | 💰 ¥800 ⏱️ 3小时 |
| "今晚8点前交稿" | ⏰ 截止20:00 |
| "帮我拆解这个任务" | 🤖 AI生成子步骤 |

---

## ⏰ 拖延监控流程

```
任务时间到 → 倒计时2分钟 → 完成启动步骤
                ↓
        [超时] 扣5金币 → 重新倒计时 → 循环
                ↓
        [完成] 获得3金币 ✅
```

**预警**：结束前20秒提醒你

---

## 📉 低效率监控流程

```
选择任务监控 → 计时1小时 → 完成当前步骤
                ↓
        [30分钟] 半程提醒
                ↓
        [超时] 扣3金币 + 显示辅助选项
                ↓
        [完成] 获得2金币 → 下一步骤
```

**辅助选项**：AI拆解 / 正念呼吸 / 切换任务
