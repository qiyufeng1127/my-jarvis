## 👋 认识 Kiki 宝宝

### Kiki 是谁？

**Kiki** 是 ManifestOS 的智能语音助手，她可以：
- 🎤 听懂你的语音指令
- 🗣️ 用语音回复你
- 🤖 帮你管理任务
- 💡 提供成长建议
- 💪 给你情感支持

### Kiki 的特点

- **自然交互**: 像和朋友聊天一样自然
- **即时响应**: 快速理解和执行指令
- **情感支持**: 在你需要时给予鼓励
- **智能理解**: 理解多种表达方式

## 🎯 唤醒 Kiki

### 找到 Kiki 按钮

**位置**: 屏幕右下角

**外观**:
- 🎤 圆形按钮
- 渐变色背景（蓝紫色）
- 悬停时会放大

### 唤醒步骤

#### 步骤 1: 点击按钮

**操作**:
- 点击右下角的 🎤 按钮

**效果**:
- 按钮变为绿色
- 按钮放大
- 右上角出现绿色指示点

#### 步骤 2: 授权麦克风

**首次使用时**:
- 浏览器会弹出麦克风权限请求
- 点击 **"允许"**

**权限说明**:
- 仅在点击按钮时使用麦克风
- 不会后台录音
- 数据不会上传到服务器

#### 步骤 3: 听到欢迎语

**Kiki 会说**:
> "我在，请说"

**状态指示**:
- 🎤 按钮绿色 + 脉冲动画
- 表示正在监听
- 可以开始说话了

## 🗣️ 与 Kiki 对话

### 对话流程

```
1. 点击 🎤 按钮
   ↓
2. 听到 "我在，请说"
   ↓
3. 说出你的指令
   ↓
4. Kiki 识别并执行
   ↓
5. Kiki 语音回复
   ↓
6. 自动关闭（或继续对话）
```

### 说话技巧

**建议**:
- 🎯 **清晰发音**: 不要太快或太慢
- 📢 **适中音量**: 不要太大声或太小声
- 🔇 **安静环境**: 避免嘈杂环境
- ⏱️ **简洁明确**: 一次说一个指令

**示例**:
```
✅ 好："查看今天的任务"
❌ 不好："嗯...那个...我想看一下...今天有什么任务来着"
```

## 🔧 语音设置

### 麦克风权限

**检查权限**:
1. 浏览器地址栏左侧
2. 点击锁图标
3. 查看麦克风权限

**重新授权**:
1. 刷新页面
2. 再次点击 Kiki 按钮
3. 点击"允许"

### 语音速度

**调整方式**:
1. 点击右上角 ⚙️ 设置
2. 选择"语音设置"
3. 调整语音速度（0.5x - 2.0x）

**建议**:
- 默认：1.0x
- 快速：1.2x
- 慢速：0.8x

### 语音音量

**调整方式**:
1. 系统音量控制
2. 或在设置中调整

## 💡 使用技巧

### 1. 最佳使用场景

**适合使用语音**:
- 🚗 开车时
- 🍳 做饭时
- 🏃 运动时
- 🛏️ 躺着时
- 💼 双手忙碌时

**不适合使用语音**:
- 🤫 公共场合（安静环境）
- 👥 开会时
- 📚 图书馆
- 🌙 深夜（影响他人）

### 2. 提高识别准确率

**技巧**:
- 使用标准普通话
- 避免方言和口音过重
- 说话清晰，不要含糊
- 避免背景噪音
- 使用常见表达方式

### 3. 快速指令

**常用指令记忆**:
```
任务：创建任务、查看任务、完成任务
成长：我的成长、查看执行力
金币：查看金币、金币余额
情感：给我鼓励、我很累
```

### 4. 连续对话

**技巧**:
- 一次对话完成后
- 可以立即再次点击按钮
- 继续下一个指令
- 无需等待

## 🐛 常见问题

### Q1: Kiki 听不到我说话？

**解决方法**:
1. 检查麦克风权限
2. 检查麦克风是否正常工作
3. 调整麦克风音量
4. 尝试更换浏览器

### Q2: Kiki 识别不准确？

**解决方法**:
1. 说话更清晰
2. 减少背景噪音
3. 使用标准普通话
4. 使用更简单的表达

### Q3: Kiki 没有声音？

**解决方法**:
1. 检查系统音量
2. 检查浏览器音量
3. 检查是否静音
4. 刷新页面重试

### Q4: 按钮点击没反应？

**解决方法**:
1. 检查浏览器兼容性（需要 Chrome/Edge）
2. 检查是否授权麦克风
3. 刷新页面
4. 清除浏览器缓存
