## 🐛 故障排除

### 如果AI没有智能回复

**检查1：API Key配置**
- 打开设置 → AI配置
- 确认DeepSeek API Key已配置
- 测试连接是否正常

**检查2：浏览器控制台**
- 按F12打开控制台
- 查看是否有错误信息
- 搜索"🧠 [智能AI]"日志

**检查3：降级模式**
- 如果看到"降级到传统模式"
- 说明AI调用失败
- 会使用原有的关键词匹配

### 如果回复太机械

**调整AI性格：**
1. 点击AI头像
2. 降低"正式程度"滑块
3. 调整"毒舌程度"
4. 保存设置

### 如果识别不准确

**优化输入：**
- 使用更自然的口语表达
- 提供更多上下文信息
- 多轮对话让AI理解
