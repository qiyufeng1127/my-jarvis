## 📱 手机端使用方法

### 方式1：从首页添加
1. 打开应用，点击底部导航栏的 **🏠 首页**
2. 在首页会看到所有可用功能的图标
3. 点击对应图标：
   - **🐾 宠物** → 打开宠物商店，可以领养宠物
   - **🎯 专注** → 进入专注模式页面
   - **⚠️ 坏习惯** → 查看坏习惯追踪
   - **🏆 排行榜** → 查看排行榜和成就

### 方式2：添加到底部导航栏
1. **长按**底部导航栏任意位置
2. 在弹出的编辑界面中，点击"添加功能"
3. 选择要添加的功能（宠物、专注、坏习惯、排行榜）
4. 拖拽调整顺序
5. 点击"保存"

## 🎮 功能说明

### 🐾 宠物系统
**入口**：首页点击 🐾 图标 或 添加到导航栏

**功能**：
- 领养6种宠物（仓鼠、兔子、小鸟、小猫、小狗、神龙）
- 喂食、玩耍、治疗
- 宠物升级提供金币加成（5%-50%）
- 完成任务自动获得经验

**价格**：
- 仓鼠：500金币（+5%加成）
- 兔子：800金币（+5%加成）
- 小鸟：600金币（+8%加成）
- 小猫：1000金币（+10%加成）
- 小狗：1000金币（+10%加成）
- 神龙：5000金币（+50%加成）⭐推荐

### 🎯 专注模式
**入口**：首页点击 🎯 图标 或 添加到导航栏

**功能**：
- 🍅 番茄钟：25分钟工作 + 5分钟休息
- 🧘 深度专注：90分钟深度工作
- 🌊 心流模式：120分钟沉浸式工作
- 完成奖励：每分钟5金币+10经验（完成+50%）
- 专注统计：今日进度、总时长、完成率

### ⚠️ 坏习惯追踪
**入口**：首页点击 ⚠️ 图标 或 添加到导航栏

**功能**：
- 自动检测7种坏习惯
- 金币惩罚（10-100金币）
- 21天矫正挑战（完成奖励500金币）
- 趋势分析（改善中/稳定/恶化）

**检测类型**：
- ⏰ 拖延症：任务超期
- 📱 分心：专注中途退出
- 🎯 完美主义：反复修改
- 🔀 多任务：同时3个以上任务
- 🌙 熬夜：23:00后工作
- 😫 不休息：连续工作超2小时
- 📋 过度承诺：待办超20个

### 🏆 排行榜
**入口**：首页点击 🏆 图标 或 添加到导航栏

**功能**：
- 5种排行榜（金币、连胜、专注、任务、宠物）
- 17个成就（普通/稀有/史诗/传说）
- 前三名特殊展示
- 自动成就检测

## 💡 使用技巧

### 最大化收益
1. **养神龙** - 50%金币加成（需要5000金币）
2. **保持连击** - 连续完成任务最高3倍奖励
3. **使用专注模式** - 额外金币和经验
4. **避免坏习惯** - 减少金币损失

### 快速升级宠物
1. **多完成任务** - 每个任务至少20经验
2. **使用专注模式** - 每分钟10经验
3. **完成长任务** - 经验与时长成正比

### 解锁成就
1. **每日完成任务** - 解锁连胜成就
2. **累计专注时长** - 解锁专注成就
3. **完成挑战** - 解锁特殊成就
