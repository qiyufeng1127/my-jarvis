# 副业赚钱追踪组件 - 需求文档

## 📋 项目概述

### 组件名称
**副业追踪器 (Side Hustle Tracker)** / **赚钱仪表盘 (Money Dashboard)**

### 核心目标
帮助自由职业者全面追踪多个副业的收入、支出、时间投入，通过数据分析和 AI 洞察，优化副业选择和时间分配，提升赚钱效率。

### 目标用户
- 自由职业者
- 多副业并行的工作者
- 需要数据驱动决策的人

---

## 🎯 核心功能需求

### 1. 副业管理

#### 1.1 副业列表
- **活跃副业**：正在进行的副业（持续性，无结束日期）
- **副业想法池**：未启动的副业想法

#### 1.2 副业信息
每个副业包含以下信息：
- 副业名称（例如：ins穿搭账号、照相馆小红书、抖音账号等）
- 副业图标/颜色
- 开始日期（可选）

#### 1.3 副业想法池
- 快速添加副业想法
- AI 分析副业可行性
- AI 评估预期收益
- AI 建议启动时机
- 一键转为活跃副业

---

### 2. 财务追踪

#### 2.1 收入记录
- **总收入**：所有副业累计收入
- **副业收入**：每个副业的收入明细
- **收入时间线**：按时间查看收入趋势
- **快速记录**：快速添加收入金额和备注

#### 2.2 支出记录
- **总支出**：所有副业累计支出
- **副业支出**：每个副业的支出明细
- **支出时间线**：按时间查看支出趋势
- **快速记录**：快速添加支出金额和备注

#### 2.3 负债统计
- **欠债金额**：当前欠了多少钱

---

### 3. 时间追踪

#### 3.1 时间投入统计
- **总时间**：所有副业累计时间
- **副业时间**：每个副业的时间明细
- **时间分布**：
  - 按副业分布
  - 按日期分布
  - 按时间段分布

#### 3.2 时间来源（核心功能）
- **自动统计**：从待办任务自动统计时间
  - 待办任务可以关联到副业
  - 任务完成时自动累计到副业时间
  - 实时更新副业时间统计
- **关联任务**：显示关联的任务列表
- **时间明细**：查看每天在每个副业上花了多少时间

---

### 4. 效率分析

#### 4.1 副业效率排名
按以下维度排名：
1. **时薪排名**：收入 / 时间投入
2. **ROI 排名**：(收入 - 支出) / 支出
3. **利润排名**：收入 - 支出
4. **增长率排名**：本月收入 / 上月收入

#### 4.2 可视化对比
- **双色柱状图**：
  - 绿色：时间投入
  - 粉色：收入金额
- **气泡图**：
  - X轴：时间投入
  - Y轴：收入
  - 气泡大小：利润
- **趋势图**：
  - 收入趋势
  - 支出趋势
  - 利润趋势

#### 4.3 效率指标
- **平均时薪**：总收入 / 总时间
- **最高时薪副业**
- **最低时薪副业**
- **时间浪费警告**：时间多但收入少的副业

---

### 5. AI 智能洞察

#### 5.1 每日洞察
- **今日总结**：今天的收入、支出、时间
- **副业建议**：建议优先做哪个副业
- **时间优化**：建议调整时间分配
- **风险提醒**：低效副业警告

#### 5.2 副业分析
- **可行性分析**：分析副业想法的可行性
- **收益预测**：预测副业可能的收入
- **竞争分析**：分析市场竞争情况
- **资源需求**：分析需要的资源和时间

#### 5.3 优化建议
- **副业优先级**：建议优先做哪些副业
- **时间分配**：建议如何分配时间
- **成本优化**：建议如何降低成本
- **收入提升**：建议如何提升收入

---

### 6. 数据可视化

#### 6.1 仪表盘
- **总资产卡片**：显示当前总资产
- **本月收入卡片**：显示本月收入
- **本月支出卡片**：显示本月支出
- **本月利润卡片**：显示本月利润

#### 6.2 图表
- **收入趋势图**：折线图
- **支出趋势图**：折线图
- **项目对比图**：柱状图
- **时间分布图**：饼图
- **效率排名图**：横向柱状图

#### 6.3 热力图
- **赚钱热力图**：哪些时间段赚钱最多
- **项目热力图**：哪些项目最活跃

## 🔗 与其他组件的交互

### 1. 与待办任务/时间轴的交互（核心交互）

#### 1.1 自动统计时间
- 待办任务可以关联到副业
- 任务完成时自动累计时间到对应副业
- 在任务卡片上显示副业标签

#### 1.2 任务关联流程
- 创建任务时可以选择关联副业
- 任务卡片显示副业图标和颜色
- 完成任务时自动更新副业时间统计
- 在副业详情中查看关联的任务列表

#### 1.3 数据同步
- 待办任务完成 → 更新副业时间
- 副业收入记录 → 更新金币系统
- 副业里程碑 → 触发成长值奖励

---

### 2. 与长期目标的交互（核心交互）

#### 2.1 自动创建目标

- 副业收入目标 → 长期目标
- 副业时间目标 → 长期目标

#### 2.2 进度同步
- 副业收入更新 → 目标进度更新
- 副业达成里程碑 → 目标完成
- 目标分析 → 副业分析

#### 2.3 目标关联
- 在目标详情中查看关联的副业
- 在副业详情中查看关联的目标
- 目标和副业双向绑定

---

### 3. 与金币系统的交互

#### 3.1 收入转化
- 副业收入 → 金币奖励
- 完成副业里程碑 → 金币奖励
- 高效副业 → 额外金币奖励

#### 3.2 激励机制
- 达到收入目标 → 金币奖励
- 提升副业效率 → 金币奖励
- 连续赚钱天数 → 金币奖励

---

### 4. 与成长系统的交互

#### 4.1 成长维度
- **创业能力**：新增成长维度
- **赚钱能力**：新增成长维度
- **副业管理**：新增成长维度

#### 4.2 成长值计算
- 副业收入 → 创业能力成长值
- 副业达成里程碑 → 副业管理成长值
- 高效副业 → 额外成长值

#### 4.3 等级提升
- 创业等级：根据总收入提升
- 副业等级：根据副业数量提升
- 效率等级：根据平均时薪提升

## 📊 数据结构设计

### 1. 副业数据结构

```typescript
interface SideHustle {
  id: string;
  userId: string;
  name: string; // 副业名称，例如：ins穿搭账号
  icon: string; // 图标
  color: string; // 颜色
  
  // 时间信息
  startDate?: Date; // 开始日期（可选）
  totalHours: number; // 总时长（小时）
  
  // 财务信息
  totalIncome: number; // 总收入
  totalExpense: number; // 总支出
  profit: number; // 利润 = 收入 - 支出
  
  // 效率指标
  hourlyRate: number; // 时薪 = 收入 / 时长
  roi: number; // ROI = (收入 - 支出) / 支出
  
  // 目标关联
  goalId?: string; // 关联的长期目标ID
  
  // AI 分析
  aiAnalysis?: {
    feasibility: number; // 可行性评分 0-100
    expectedIncome: number; // 预期收入
    recommendation: string; // AI 建议
    risks: string[]; // 风险提示
  };
  
  status: 'active' | 'idea'; // 活跃 | 想法
  createdAt: Date;
  updatedAt: Date;
}
```

### 2. 收入记录

```typescript
interface IncomeRecord {
  id: string;
  sideHustleId: string; // 关联的副业ID
  amount: number; // 金额
  description?: string; // 备注
  date: Date; // 日期
  createdAt: Date;
}
```

### 3. 支出记录

```typescript
interface ExpenseRecord {
  id: string;
  sideHustleId: string; // 关联的副业ID
  amount: number; // 金额
  description?: string; // 备注
  date: Date; // 日期
  createdAt: Date;
}
```

### 4. 时间记录（自动统计）

```typescript
interface TimeRecord {
  id: string;
  sideHustleId: string; // 关联的副业ID
  duration: number; // 时长（分钟）
  date: Date; // 日期
  taskId: string; // 关联的待办任务ID
  taskTitle: string; // 任务标题
  createdAt: Date;
```

### 5. 负债记录

```typescript
interface DebtRecord {
  id: string;

  description: string; // 描述
  dueDate?: Date; // 还款日期
  isPaid: boolean; // 是否已还

}
```
