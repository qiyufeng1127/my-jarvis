# 📔 日记系统功能说明

## 功能概述

全新的日记系统已经集成到全景记忆模块中，提供三种不同类型的日记分析和多视角的AI对话功能。

## 访问方式

### 方式1：通过全景记忆模块
1. 在主控面板中找到"全景记忆栏"模块
2. 点击右上角的"日记系统"按钮
3. 进入日记系统界面

### 方式2：直接访问测试页面
访问 `/diary-test` 路径查看日记系统的独立测试页面

## 三种日记类型

### 1. 📋 内容结构分析日记
**功能特点：**
- 梳理当天时间轴上的所有任务、备注、赚钱记录、心情记录
- 按时间脉络整理成清晰的条目日记
- 包含：核心事件、时间场景、具体经过、核心备注

**使用场景：**
- 每日复盘，回顾一天的工作和生活
- 整理思绪，看清一天的时间分配
- 发现时间管理的问题和改进空间

### 2. 💗 情绪链条日记
**功能特点：**
- 识别当天时间链条中的所有情绪（焦虑、委屈、开心、烦躁等）
- 标注每种情绪对应的触发点
- 区分表层情绪和深层情绪
- 总结情绪变化规律和未被满足的需求

**使用场景：**
- 情绪管理和自我觉察
- 理解情绪背后的真实需求
- 发现情绪模式，避免重复陷入负面情绪

### 3. ⭐ 成功日记
**功能特点：**
- 记录每天的高光时刻和成就（任何小事都算）
- 放大优势，积累自信
- 如果不确定算不算成功，那就一定算！

**使用场景：**
- 建立自信，看到自己的成长
- 睡前复盘，带着成就感入睡
- 低谷期翻看，找回力量

## AI功能按钮

### 🎯 教练给的小建议
**提示词逻辑：**
- 结合当天的核心需求
- 给出5条以内具体可立即执行的小建议
- 拒绝空泛的大道理
- 帮助缓解焦虑，兼顾情绪和问题解决

**适用场景：**
- 感到迷茫时，需要具体的行动指引
- 焦虑时，需要可执行的缓解方案
- 想要改变但不知从何开始

### 📊 情绪复盘
**提示词逻辑：**
- 调取历史情绪记录（最近30天）
- 识别触发场景、事件、表层深层情绪、核心诉求
- 对比当前情绪日记，列出核心重合点与差异点
- 发现情绪模式和成长变化

**适用场景：**
- 反复出现相似的情绪困扰
- 想要理解自己的情绪模式
- 看到自己在情绪管理上的成长

### ✨ 理想的自己（成功日记专属）
**五种视角：**

1. **1年后理想的自己** 🙋
   - 看看一年后的你会怎么说
   - 短期目标的视角

2. **5年后理想的自己** 👑
   - 从更长远的角度看待当下
   - 人生大局的视角

3. **旁观者中立视角** 👁️
   - 跳出局限，客观看待问题
   - 理性分析的视角

4. **上帝全局视角** 🌍
   - 从全局视角理解人生
   - 超越个人得失的视角

5. **偶像眼中的自己** 💖
   - 用榜样的智慧指引方向
   - 激励和启发的视角

**提示词逻辑：**
- AI扮演选定的角色/视角
- 分析当前面临的主要问题和挑战
- 指出优势和已经取得的进步
- 给出具体、温暖、有方向感的建议
- 帮助看到更大的可能性
- 给予继续前进的力量和信心

## 使用流程

### 基础流程
1. **选择日记类型** - 在顶部选择三种日记类型之一
2. **查看日历** - 日历上会显示有记录的日期（带彩色圆点）
3. **点击日期** - 选择要查看的日期
4. **查看原始数据** - 系统会展示当天的所有记录和任务
5. **生成日记** - 点击"生成日记"按钮，AI会分析并生成日记
6. **获取建议** - 点击"教练给的小建议"或"情绪复盘"获取深度分析

### 成功日记特殊流程
1. 选择"成功日记"类型
2. 查看日历，选择日期
3. 生成成功日记
4. 点击"理想的自己"按钮
5. 选择一个视角（1年后、5年后、旁观者等）
6. 点击"开始对话"，获得来自不同视角的建议和支持

## 日历功能

### 视觉标识
- **彩色圆点**：表示该天有记录
- **数字标记**：显示记录数量（超过1条时显示）
- **边框高亮**：今天的日期
- **背景高亮**：选中的日期

### 颜色含义
- **蓝色** (#3B82F6)：内容结构分析日记
- **粉色** (#EC4899)：情绪链条日记
- **橙色** (#F59E0B)：成功日记

## 数据来源

日记系统会自动整合以下数据：
- ✅ 任务记录（从任务系统）
- 💭 心情记录（从全景记忆）
- 📝 碎碎念（从全景记忆）
- ✅ 待办事项（从全景记忆）
- ⭐ 成功记录（从全景记忆）
- 🙏 感恩记录（从全景记忆）

## AI配置要求

日记系统需要配置AI服务才能使用：
1. 进入"设置"模块
2. 配置AI API Key和端点
3. 默认使用DeepSeek API（已预配置）

## 技术特点

- ✅ 完全本地化存储，数据安全
- ✅ 实时同步，无需刷新
- ✅ 响应式设计，支持移动端
- ✅ 智能日历，自动标记有记录的日期
- ✅ 多视角AI对话，提供全方位支持
- ✅ 温暖的语言风格，注重用户体验

## 使用建议

### 每日使用
1. **早上**：查看昨天的内容结构分析日记，规划今天
2. **晚上**：写成功日记，记录3件高光时刻
3. **情绪波动时**：记录情绪，使用情绪链条日记分析

### 定期复盘
1. **每周**：查看本周的情绪链条日记，发现情绪模式
2. **每月**：使用"理想的自己"功能，从不同视角审视自己
3. **低谷期**：翻看成功日记，找回自信和力量

### 最佳实践
- 不要追求完美，任何小事都值得记录
- 情绪记录要真实，不要美化或压抑
- 成功日记要具体，记录细节而非笼统的"今天不错"
- 定期使用"情绪复盘"，看到自己的成长
