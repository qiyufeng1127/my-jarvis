## 🚀 快速开始

### 1. 安装
```bash
npm install
```

### 2. 配置
```bash
cp .env.example .env
# 编辑 .env 填入 Supabase 配置
```

### 3. 数据库
在 Supabase 控制台执行 `supabase/schema.sql`

### 4. 启动
```bash
npm run dev
```
