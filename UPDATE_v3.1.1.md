## 🔄 待完成的优化

### 需要手动修改 app.js

由于 `app.js` 文件过大（3872行），需要手动进行以下修改：

#### 1. 找到拖延监控设置渲染代码
搜索关键词：`renderProcrastinationMonitor` 或 `拖延监控设置`

#### 2. 重新组织设置项布局
按照 `SETTINGS_UI_OPTIMIZATION.md` 中的示例代码，将：
```
启动宽限期: [输入框]
└─ 自定义预警提示语: [输入框]

预警提前时间: [输入框]
└─ 自定义超时提示语: [输入框]
```

#### 3. 找到低效率监控设置渲染代码
搜索关键词：`renderInefficiencyMonitor` 或 `低效率监控设置`

#### 4. 添加自定义提示语输入框
在"判定时长"设置下方添加：
```
判定时长: [输入框]
└─ 自定义低效率提示语: [输入框]
```

#### 5. 添加 CSS 样式
将 `SETTINGS_UI_OPTIMIZATION.md` 中的 CSS 代码添加到 `styles/main.css`
