# 🇨🇳 中国大陆访问解决方案

## 问题分析

你的网站需要翻墙才能访问，主要原因是：

1. **Vercel 默认域名被墙**: `*.vercel.app` 域名在中国大陆被防火墙屏蔽
2. **服务器位置**: Vercel 默认使用美国服务器，访问速度慢

## ✅ 已完成的优化

### 1. 修改 Vercel 配置
已在 `vercel.json` 中添加：
- `"regions": ["hkg1", "sin1"]` - 使用香港和新加坡服务器（离中国最近）
- 添加 CORS 头部，确保跨域访问正常

## 🚀 推荐解决方案（按优先级）

### 方案 1：绑定自定义域名（最佳方案）⭐⭐⭐⭐⭐

**为什么这是最佳方案？**
- ✅ Vercel 支持免费绑定自定义域名
- ✅ 自定义域名不会被墙（除非你的域名本身有问题）
- ✅ 可以使用国内 DNS 加速
- ✅ 看起来更专业

**操作步骤：**

1. **购买域名**（推荐国内平台）
   - 阿里云：https://wanwang.aliyun.com/
   - 腾讯云：https://dnspod.cloud.tencent.com/
   - 推荐域名：`manifestos.cn` 或 `manifestos.com`

2. **在 Vercel 绑定域名**
   ```
   1. 进入 Vercel 项目设置
   2. 点击 "Domains"
   3. 添加你的域名（例如：www.manifestos.cn）
   4. Vercel 会给你 DNS 记录
   ```

3. **配置 DNS**（在域名提供商后台）
   ```
   类型: CNAME
   主机记录: www
   记录值: cname.vercel-dns.com
   ```

4. **等待生效**（通常 5-30 分钟）

**成本：** 域名费用约 50-100 元/年

### 方案 4：使用反向代理 ⭐⭐

如果暂时不想换平台或买域名，可以使用反向代理：

#### A. Cloudflare Workers（免费）
```javascript
// 在 Cloudflare Workers 中创建
addEventListener('fetch', event => {
  event.respondWith(handleRequest(event.request))
})

async function handleRequest(request) {
  const url = new URL(request.url)
  url.hostname = 'your-app.vercel.app' // 替换为你的 Vercel 域名
  
  return fetch(url, {
    method: request.method,
    headers: request.headers,
    body: request.body
  })
}
```

然后绑定一个 Cloudflare 的域名（如 `your-app.workers.dev`）

#### B. 使用国内服务器做反向代理
- 租一个国内服务器（阿里云/腾讯云）
- 使用 Nginx 反向代理到 Vercel

## ❓ 常见问题

**Q: 为什么 Vercel 会被墙？**
A: 因为 `*.vercel.app` 域名被中国防火墙屏蔽，但绑定自定义域名后就不会被墙。

**Q: 域名多少钱？**
A: 国内域名（.cn）约 50 元/年，国际域名（.com）约 60-100 元/年。

**Q: 需要备案吗？**
A: 如果使用 Vercel + 自定义域名，不需要备案。只有使用国内服务器才需要备案。

**Q: 多久能解决？**
A: 购买域名并配置，最快 30 分钟就能让中国用户访问。
