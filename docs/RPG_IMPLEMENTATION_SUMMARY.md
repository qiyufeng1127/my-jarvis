#### P0-5: 雷达图实时更新
**文件：** `src/services/rpgRadarUpdater.ts`

**实现内容：**
- ✅ **任务完成时更新正向能力**：
  - 基础更新：所有任务 +3 执行力
  - 任务类型加成：
    - work → 赚钱能力+2，专注力+2
    - study → 学习力+3，专注力+2
    - exercise → 健康+4，执行力+2
    - social → 社交力+3，情绪管理+2
    - creative → 创造力+3，学习力+1
  - 难度加成：easy×1.0, medium×1.2, hard×1.5
  - 效率加成：效率≥80% → 专注力+2，执行力+1
  - 改进任务加成：执行力+3，情绪管理+2
- ✅ **改进任务完成时降低负向行为**：
  - 拖延相关 → 拖延-10，逃避-5
  - 效率相关 → 分心-10，拖延-3
  - 一致性相关 → 拖延-8，自我怀疑-5
  - 焦虑相关 → 焦虑-10，消极-5
- ✅ **任务失败时增加负向行为**：
  - 超时 → 拖延+5，逃避+3
  - 低效 → 分心+5，焦虑+3
- ✅ **历史数据重新计算**：
  - 基于最近30天任务
  - 根据任务类型分布计算能力值
  - 根据效率和拖延情况调整
