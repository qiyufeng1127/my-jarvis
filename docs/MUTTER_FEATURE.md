# 碎碎念功能实现文档

## 功能概述

将原来的"查看进度"按钮改造成"碎碎念"按钮，用户可以随时记录心情、想法和碎碎念，AI会智能分析并保存到时间轴和记忆库。

## 实现的功能

### 1. 快速入口
- **位置**: AI聊天界面底部快速指令区
- **按钮**: 💭 碎碎念
- **样式**: 复古糖果色，点击后紫色高亮

### 2. 碎碎念模式
点击按钮后进入碎碎念模式，AI会提示：
```
💭 **碎碎念模式已开启**

现在你可以随意说说你的想法、心情或任何碎碎念，例如：

• "今天好累啊，感觉什么都不想做"
• "刚才那个会议真的让我很焦虑"
• "突然想起小时候的事情，有点想家"
• "今天天气真好，心情也变好了"

AI会智能分析：
✅ 你的情绪状态
✅ 自动分类和打标签
✅ 生成一句话总结
✅ 保存到时间轴和记忆库

💡 说完后我会帮你记录下来~
```

### 3. AI智能分析
用户输入碎碎念后，AI会分析：
- **情绪识别**: 识别具体情绪（如：开心、焦虑、平静、兴奋等）
- **情绪Emoji**: 自动匹配对应的emoji（如：😊、😰、😌、🤩等）
- **智能分类**: 自动分类（工作、生活、学习、情感、健康等）
- **一句话总结**: 提炼核心内容（20字以内）
- **智能标签**: 自动生成3个相关标签

### 4. 多重保存
分析完成后，碎碎念会被保存到：

#### 4.1 记忆库
- 类型: `mutter`
- 包含完整内容、情绪、标签
- 可在记忆库模块查看

#### 4.2 时间轴
创建一个醒目的记录卡片：
- **标题**: `💭{emoji} {一句话总结}`
- **描述**: 完整内容 + 情绪 + 分类 + 标签
- **颜色**: 复古糖果薰衣草紫 (#DDA0DD)
- **状态**: 已完成（不需要执行）
- **标签**: `💭碎碎念` + AI生成的标签
- **时长**: 1分钟（仅作记录）

### 5. 奖励机制
- **金币**: +30 金币
- **成长值**: 自动计算

### 6. 反馈展示
AI会返回详细的分析结果：
```
✅ **碎碎念已记录！**

💭 内容: {用户输入}

😊 **心情**: 开心
📂 **分类**: 生活
✨ **总结**: 今天天气真好
🏷️ **标签**: 心情、天气、生活

💰 获得 30 金币！

💡 已同步到记忆库和时间轴~
```

#### 2. MutterModal.tsx（新建）
- 独立的碎碎念模态框组件
- 复古糖果色设计风格
- AI分析和结果展示

## 使用流程

1. 用户点击 "💭 碎碎念" 按钮
2. AI提示进入碎碎念模式
3. 用户输入任何想法、心情或碎碎念
4. AI智能分析情绪、分类、生成标签
5. 保存到记忆库和时间轴
6. 显示分析结果和奖励
7. 自动退出碎碎念模式

## 设计特点

### 复古糖果色风格
- **薰衣草紫**: #DDA0DD（时间轴卡片背景）
- **糖果粉**: #FF6B9D（按钮和强调色）
- **温暖友好**: 使用emoji和温暖的文案

### 用户体验
- **零门槛**: 随时随地记录，无需格式
- **智能化**: AI自动分析，无需手动分类
- **可视化**: 时间轴上清晰展示
- **激励性**: 金币奖励，鼓励记录

## 未来优化方向

1. **情绪趋势分析**: 统计一段时间的情绪变化
2. **智能提醒**: 根据情绪状态给出建议
3. **情绪日历**: 可视化展示每天的情绪
4. **关联分析**: 分析情绪与任务完成度的关系
5. **导出功能**: 导出碎碎念日记

## 注意事项

- 需要配置AI才能使用智能分析功能
- 如果AI分析失败，会使用默认值保存
- 碎碎念会占用时间轴1分钟的位置
- 自动标记为已完成状态，不影响任务统计
