# 新功能快速上手指南 🚀

## 🔔 浏览器通知

### 如何启用？
1. 打开应用时会自动弹出权限请求
2. 点击"允许"即可

### 什么时候会收到通知？
- ⏰ **任务开始时**：提醒你任务已开始，需要启动验证
- ⏰ **任务即将结束**：提前2分钟提醒，准备完成验证
- ⏰ **任务结束时**：提醒你任务已结束，需要完成验证

### 通知内容示例：
```
📋 任务开始
上厕所 现在已开始，请进行启动验证哦！
```

## 🎤 免手模式（语音控制）

### 如何打开？
1. 点击右下角**紫色喇叭按钮**（在AI按钮上方）
2. 点击麦克风按钮开始监听
3. 说出你的指令

### 支持的指令：
| 指令 | 功能 |
|------|------|
| "现在正在做什么任务" | 查询当前任务 |
| "下一个任务是什么" | 查询下一个任务 |
| "这个任务过去了多长时间" | 查询任务进度 |
| "下个任务还有多长时间开始" | 查询倒计时 |
| "删除今天的任务" | 批量删除今日任务 |
| "帮我安排任务" | 调用AI助手 |

### 使用示例：
```
你说："现在正在做什么任务"
AI回复："当前正在进行上厕所，已经进行了5分钟"
```

## ✅ 启动验证状态

### 新变化：
- 完成启动验证后，按钮会变成 **"✅已启动"**
- 背景变为**绿色半透明**
- 不能重复进行启动验证

### 视觉效果：
```
未验证：[*start] (白色背景)
已验证：[✅已启动] (绿色背景)
```

## 🎯 快速测试

### 1分钟测试通知功能：
```
1. 创建任务"测试"
2. 设置开始时间为1分钟后
3. 等待通知弹出
```

### 1分钟测试语音控制：
```
1. 点击紫色喇叭按钮
2. 点击麦克风
3. 说："现在正在做什么任务"
4. 听AI回复
```

### 1分钟测试验证状态：
```
1. 创建任务，启用验证
2. 点击*start，拍照验证
3. 查看按钮变成"✅已启动"
```

## 💡 小贴士

### 通知
- 通知会自动消失（5秒后）
- 点击通知可以聚焦窗口
- 支持语音提示音

### 语音控制
- 需要使用**Chrome浏览器**（最佳支持）
- 说话要清晰，避免环境噪音
- 可以随时点击麦克风停止监听

### 验证系统
- 照片要清晰，光线充足
- 物品要在照片中清晰可见
- 查看调试信息了解失败原因
