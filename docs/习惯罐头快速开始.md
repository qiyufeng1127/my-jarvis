# 习惯罐头系统 - 快速开始

## 🎯 5分钟上手指南

### 第一步：打开习惯模块

**电脑端**
1. 在主面板中找到「坏习惯」模块（⚠️ 图标）
2. 点击进入

**手机端**
1. 点击底部导航栏的「更多」
2. 选择「习惯」（⚠️ 图标）

### 第二步：了解预设习惯

系统已经为你准备了 6 个常见坏习惯：

| Emoji | 名称 | 说明 |
|-------|------|------|
| 🌙 | 熬夜 | 最后一个任务晚于 23:00 |
| 🛌 | 晚起 | 第一个任务晚于 10:30 |
| 🕒 | 拖延 | 任务启动超时 |
| 🐢 | 低效率 | 任务完成超时 |
| 🍱 | 点外卖 | 任务包含外卖关键词 |
| 🥣 | 不吃午饭 | 11:30-13:00 无午饭任务 |

这些习惯会**自动监控**你的时间轴事件！

### 第三步：查看今天的罐头

1. 在日历上找到今天的日期（有高亮边框）
2. 看看罐头里有没有 emoji
3. 点击罐头查看详细记录

**罐头颜色含义：**
- 🟢 绿色 = 今天很棒，没有坏习惯！
- 🟡 黄色 = 有 1-3 个坏习惯
- 🔴 红色 = 超过 3 个坏习惯，需要注意了

### 第四步：调整规则（可选）

如果你觉得默认规则不适合你：

1. 点击右上角「规则设置」按钮
2. 找到要调整的习惯
3. 点击「编辑规则」
4. 修改时间阈值或关键词
5. 保存

**常见调整：**
- 熬夜时间改为 24:00（夜猫子专用）
- 晚起时间改为 09:00（早起鸟专用）
- 添加更多外卖关键词（如「点餐」「送餐」）

### 第五步：创建自己的习惯

想追踪其他坏习惯？

1. 点击「自定义坏习惯」按钮
2. 输入名称，如「刷短视频」
3. 选择 emoji，如 📱
4. 选择规则类型：
   - **关键词规则**：任务包含「抖音」「快手」等
   - **时间阈值**：某个时间段做某事
   - **手动记录**：自己手动添加
5. 点击「创建坏习惯」

## 💡 使用技巧

### 技巧 1：合理设置阈值
不要太严格，也不要太宽松。根据自己的实际情况调整。

**示例：**
- 如果你是程序员，经常工作到 23:30，可以把熬夜时间改为 00:00
- 如果你想早起，可以把晚起时间改为 08:00

### 技巧 2：关注高频习惯
每月看看「最频发坏习惯」，重点改善它。

**方法：**
1. 在日历顶部查看统计
2. 找到最频发的坏习惯
3. 思考原因并制定改善计划
4. 配合金币系统设置惩罚

### 技巧 3：手动补充记录
自动监控可能漏掉一些情况，可以手动添加。

**步骤：**
1. 点击对应日期的罐头
2. 在详情页手动添加
3. 输入原因和次数

### 技巧 4：定期回顾
每周日晚上，回顾本周的习惯罐头。

**回顾清单：**
- [ ] 哪些天是绿色罐头？
- [ ] 最常出现的坏习惯是什么？
- [ ] 有什么改善的迹象吗？
- [ ] 下周要重点改善什么？

## ❓ 常见问题

**Q: 为什么今天的罐头是空的？**
A: 可能是：
1. 今天还没有创建任务
2. 任务都很正常，没有触发坏习惯
3. 规则设置不合适

**Q: 可以删除某天的记录吗？**
A: 手动添加的可以删除，自动记录的不能删除（保证数据真实性）。

**Q: 预设习惯可以删除吗？**
A: 不能删除，但可以禁用。在规则设置中关闭开关即可。

**Q: 如何导出数据？**
A: 目前数据存储在浏览器 localStorage 中，未来会支持导出功能。
