# 任务时间自动调整功能 - README

## 🎯 一句话总结

**任务启动/完成时，时间自动调整到实际时间，冲突任务自动下移到空闲时段。**

---

## ⚡ 快速开始

### 使用方法
1. 点击任务的"启动"按钮
2. 拍照完成验证
3. ✅ 任务自动移到当前时间
4. ⚠️ 冲突任务自动下移

### 示例
```
原计划: 6:00 - 7:00 工作
实际启动: 3:00
调整后: 3:00 - 4:00 工作 ← 自动调整
        4:00 - 4:30 洗碗 ← 原3:00任务自动下移
```
