# 🎨 可自定义仪表盘使用指南

## 🎯 设计理念

**完全自由的个性化工作空间** - 像搭积木一样自由组合你需要的功能模块。

---

## ✨ 核心特性

### 1. 左侧功能模块栏 📊

**位置**: 网站最左侧的垂直栏

**包含的功能模块**:
- 📊 **总控面板** - 主仪表盘，快速查看关键数据
- 🎯 **成长系统** - 大女主成长追踪
- ✅ **任务管理** - 智能任务系统
- 💰 **金币经济** - 动态奖励系统
- ⚠️ **坏习惯** - 习惯追踪与改进
- 📈 **数据报告** - 分析与统计
- ⚙️ **设置** - 自定义与配置
- 🎤 **Kiki宝宝** - 语音助手

---

## 🎮 基础操作

### 1. 添加功能模块 ➕

**操作步骤**:
1. 在左侧功能栏找到需要的功能图标
2. **单击图标** - 模块会自动添加到主页
3. 模块以默认大小和位置出现

**效果**:
- 图标变为高亮状态（粉色背景）
- 模块出现在主页空白区域
- 默认尺寸：中等（400x300px）
- 默认位置：(100, 100)

**示例**:
```
点击 📊 → 总控面板模块出现在主页
点击 🎯 → 成长系统模块出现在主页
```

---

### 2. 隐藏功能模块 ❌

**操作步骤**:
1. 找到已添加的模块
2. 点击模块右上角的 **×** 按钮
3. 模块从主页消失

**效果**:
- 模块隐藏（不是删除）
- 左侧图标恢复正常状态
- 再次点击左侧图标可重新显示

**快捷方式**:
- 也可以直接再次点击左侧栏的图标来隐藏

---

### 3. 调整模块位置 ↔️

**操作步骤**:
1. 将鼠标移动到模块顶部（标题栏区域）
2. 看到 ⋮⋮ 拖拽图标
3. **按住并拖拽** - 模块跟随鼠标移动
4. 拖到合适位置后松开鼠标

**特点**:
- 自由拖拽，无网格限制
- 拖拽时模块层级提升
- 光标变为"抓取"状态
- 实时预览位置

**技巧**:
- 可以将相关模块放在一起
- 常用模块放在显眼位置
- 不常用模块放在边缘

---

### 4. 调整模块大小 ↘️

**方法 1: 点击切换**

**操作步骤**:
1. 将鼠标移动到模块右下角
2. 看到小方块图标
3. **点击** - 模块在三种尺寸间循环切换

**三种预设尺寸**:
- 🔹 **小** (300x200px) - 适合简单信息
- 🔸 **中** (400x300px) - 默认尺寸
- 🔶 **大** (600x400px) - 适合详细内容

**循环顺序**:
```
小 → 中 → 大 → 小 → ...
```

**方法 2: 拖拽调整（计划中）**

**操作步骤**:
1. 将鼠标移动到模块右下角
2. 鼠标变成 ↘️ 双箭头
3. **按住并拖拽** - 自由调整大小
4. 松开鼠标固定尺寸

---

### 5. 修改模块颜色 🎨

**操作步骤**:
1. 将鼠标移动到模块右上角
2. 点击 **🎨 调色板图标**
3. 弹出颜色选择器
4. 选择喜欢的颜色

**两种选择方式**:

#### 方式 1: 预设颜色

**6种预设颜色**:
- 🌸 粉色 (#DD617C)
- 🌿 绿色 (#6D9978)
- 🌻 黄色 (#E8C259)
- 🏜️ 米色 (#D1CBBA)
- 🌹 红色 (#AC0327)
- ⚫ 灰色 (#9CA3AF)

**操作**: 点击色块即可应用

#### 方式 2: 自定义颜色

**操作**:
1. 点击底部的颜色选择器
2. 选择任意颜色
3. 或输入十六进制颜色代码（如 #FF5733）

**效果**:
- 模块顶部边框颜色改变
- 模块头部背景色改变（浅色版本）
- 图标颜色改变
- 立即生效
