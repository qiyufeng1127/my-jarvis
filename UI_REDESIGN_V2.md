## 🚀 使用方法

### 查看新界面
1. 刷新浏览器 `http://localhost:3000/`
2. 默认进入首页（时间轴）
3. 可以看到新的玫粉色顶部卡片

### 上传照片
1. 点击右侧的"上传照片"区域
2. 选择 PNG/JPG 图片
3. 或直接拖拽图片到该区域

### 访问其他功能
1. 点击底部的"设置"按钮（⚙️）
2. 在弹出的网格中选择功能
3. 点击即可切换

### 返回首页
1. 点击底部中间的"首页"按钮（🏠）
2. 自动切换到时间轴
3. 显示顶部卡片

## 🔧 技术实现

### 1. 条件渲染
```tsx
{activeTab === 'timeline' && <MobileTopBar />}
```

### 2. 照片上传
```tsx
<input type="file" accept="image/png,image/jpeg" />
// 支持拖拽上传
onDragOver, onDrop
```

### 3. 渐变背景
```css
background: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
```
