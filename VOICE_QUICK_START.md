# 🎤 Kiki 语音助手 - 快速启动指南

## 🚀 立即开始

### 第一步：唤醒 Kiki
点击右下角的 **🎤 Kiki宝宝** 按钮，或直接说 **"Kiki宝宝"**

### 第二步：等待响应
- 听到 Kiki 说："**我在，请说**"
- 看到粉紫色渐变按钮和声波动画
- 注意倒计时：你有 **8秒** 时间发出指令

### 第三步：说出指令
选择以下任一指令类型：

## 💡 使用技巧

### ✅ 这样说更好
- ✓ "创建任务：下午3点写报告，需要2小时" （完整信息）
- ✓ "查看今天的任务" （清晰明确）
- ✓ "我的专注力现在多少了" （自然表达）

### ❌ 避免这样说
- ✗ "任务" （信息不完整）
- ✗ "看看" （不够具体）
- ✗ 说话太快或太小声

## 🎨 界面说明

### 按钮状态
- 🎤 **灰色图标** = 休眠中，点击唤醒
- 💜 **粉紫渐变 + 声波** = 已激活，正在聆听
- ⭕ **倒计时圆环** = 剩余时间（8秒倒数）

### 状态标签
- 🟢 **聆听中 (Xs)** = 正在接收你的语音
- 🔵 **Kiki 说话中...** = 正在播放回复
- 🟡 **准备中...** = 正在初始化

### 识别显示
- 白色卡片会实时显示你说的话
- 显示指令类型（任务/查询/控制/情感）
- 显示具体意图

## 🆘 遇到问题？

### Kiki 没有响应
1. 检查浏览器麦克风权限
2. 确保使用 Chrome 或 Edge 浏览器
3. 检查麦克风是否正常工作

### 识别不准确
1. 说话清晰，语速适中
2. 避免环境噪音
3. 靠近麦克风
4. 使用更具体的表达

### 想要取消
1. 再次点击 🎤 按钮
2. 等待 8 秒自动休眠
3. 说"取消"或"算了"
