## 快速测试

### 方法1：运行测试脚本

打开浏览器控制台（F12），复制粘贴以下代码：

```javascript
// 给已完成的任务添加测试数据
const taskStore = JSON.parse(localStorage.getItem('task-storage'));
const completedTask = taskStore.state.tasks.find(t => t.status === 'completed');

if (completedTask) {
  completedTask.startTimeoutCount = 3; // 启动超时3次
  completedTask.completeTimeoutCount = 2; // 完成超时2次
  completedTask.completionEfficiency = 45; // 效率45%
  completedTask.efficiencyLevel = 'poor';
  
  localStorage.setItem('task-storage', JSON.stringify(taskStore));
  console.log('✅ 已添加测试数据，刷新页面查看效果');
  location.reload();
} else {
  console.log('❌ 没有找到已完成的任务');
}
```

### 方法2：真实测试

1. 创建一个新任务并启用验证
2. 等待任务到时间
3. **故意延迟启动3次**（每次超过2分钟）
4. 启动任务后，**故意延迟完成2次**（每次超过时限）
5. 最后完成任务
6. 查看任务卡片右上角的按钮

## 数据说明

- **启动超时次数**：每超过2分钟启动一次，计数+1
- **完成超时次数**：每超过任务时限完成一次，计数+1
- **扣除金币**：每次超时扣除 20% 的任务金币奖励
- **完成效率**：实际拍照数 / 计划拍照数 × 100%

## 常见问题

**Q: 按钮太小看不清？**
A: 已经改大了！现在是带次数的按钮，更醒目。

**Q: 点击按钮没反应？**
A: 确保任务有超时或低效率数据。运行测试脚本添加数据。

**Q: 历史记录显示不全？**
A: 检查任务对象中是否有 `startTimeoutCount` 和 `completeTimeoutCount` 字段。

**Q: 旧任务能显示吗？**
A: 旧任务需要运行测试脚本添加数据。新任务会自动记录。
