## ✨ 快速开始

### 1️⃣ 安装依赖
```bash
npm install
```

### 2️⃣ 配置环境变量
```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，填入你的 Supabase 配置
# VITE_SUPABASE_URL=你的_supabase_url
# VITE_SUPABASE_ANON_KEY=你的_supabase_key
```

### 3️⃣ 设置数据库
在 Supabase 控制台执行 `supabase/schema.sql` 文件

### 4️⃣ 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:3000 🚀
